Quick Start

The following guides provide step-by-step instructions for an initial system setup of the AD-FMCOMMS5-EBZ on various FPGA development boards. They cover bitstream loading, booting a Linux distribution, and launching the IIO Oscilloscope for RF waveform capture and playback.
