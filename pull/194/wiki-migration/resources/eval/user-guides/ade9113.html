    Layout Guidelines ADE9113 ADE9112 ADE9103

    Overview

    These layout guidelines are written as a comprehensive guide to all the considerations that must be taken when creating a PCB (Printed Circuit Board) layout with the ADE9112/ADE9113. The guidelines use standard PCB practices but if not followed, could lead to poor performance in certain EMC (Electromagnetic Compatibility) tests. The EMC testing results will be discussed along with the layout guidelines.

    Testing was done on a board with four ADE9113s, meant to replicate a 3-phase system with isolated Neutral current measurement. Note that the EVAL-ADE9113KTZ does not have all of these layout recommendations implemented because there were constant improvements being made and updating the guidelines is the fastest way to communicate those changes. The test board also included an ADE9178 to perform the metrology calculations, specifically to get the energy pulse output for accuracy testing.

    Key Highlights:

    • A 4-layer PCB is required to pass CISPR 32 Class B emissions, and the guidelines will show the most area effective way to implement it.

    • An external oscillator (or clock from external source like a microcontroller) must be used in most cases to pass IEC 61000-4-4 EFT testing at ±4800 V, both 5 kHz and 100 kHz. In certain cases a crystal can be used and that is described in the guidelines.

    Testing Referenced in these Guidelines:

    • IEC 61000-4-3 RF IMMUNITY 30 V/m 80% 1 kHz modulation: PASSED

    • IEC 61000-4-4 EFT ± 4800 V, both 5 kHz and 100 kHz: PASSED

      • Tested both with and without PE (Protective Earth) connected to the safe-side GND, the same GND as the ADE9178, on the opposite side of the isolation barrier as the Line voltage.

      • For more details: EFT with Optimized CAP

    • CISPR 32 Class B 30 MHz to 1 GHz and 1 GHz to 6 GHz: PASSED

      • For more details: Emissions with optimized PCB Cap

    Connection Diagram

    The ADE9113/9112 SPI are connected is a Daisy Chain configuration. In this configuration the first ADE9113/9112 in the chain, the one connected to MOSI of the MCU, has the 16.384 MHz clock provided by a crystal , external OSC or MCU pin. The remaining ADE9113/9112 have the 16.384Mhz clock provided by the Clockout/Dready pin of the First ADE9113/9112 connected to the XTALIN of the remaining ADE9113/9112. Note: The schematic uses a crystal but if PE(Protective Earth) is connected to safeside ground an oscillator may required to pass EMC test. See datasheet for more information. ADE9113 Datasheet

    Figure 1: Application Circuit Diagram for a 3 Phase meter with isolated neutral

    Emissions Plots Cispr32 3 Meter Chamber

    The emission plots were captured with a 2 layer PCB without internal PCB cap. This is the worst case emissions and over the class B limit. These emissions are tested with the ferrites in isopower on each channel as in Figure 1.

    Important

    It is not recommended to use a 2 Layer PCB. 4 layer PCB with an internal PCB cap is recommended.

    Figure 2b: Vertical emissions of isolated 4 ADE9113 without PCB capacitor with ferrites on VDDISO and AGNDISO

    Layout Stackup for PCB Stitch CAP

    To improve the emissions an internal PCB cap is created using inner layers of the PCB. This capacitance has low inductance and performs better than external caps. The cap should be > than 30pf. The cap dielectric should be >0.4mm. The thicker The dielectric is the larger the cap area required. The recommendation is to keep the dielectric thickness between 0.4 and 0.5mm. These distances came from IEC 61010-1 Distance between adjacent conductors along an interface of an inner layer

    Figure 3: 4 Layer Stackup to construct PCB capacitor

    For more information see AN-0971: Recommendations for Control of Radiated Emissions with isoPower Devices

    Layout of PCB on the ADE9178 Evaluation Board

    The ADE9178 Eval board has been modified to pass EFT with + pulses on PE(Protective Earth) connected to the safe side and emissions based on the original recommendations. If PE(Protective Earth) is not used then the crystal circuit is acceptable. Using ADE9178 for EV Charger Metrology Solution

    If PE (Protective Earth) is not used: Changing the Isopower connections, removing the ferrites and connecting GNDISO to AGND and VDDISO to VDD see Figure 4. Change the power and ground connection allow for a larger PCB cap area.

    Changes if PE(Protective Earth) is used: Replacing the Crystal with an oscillator or clock from the MCU in addition to if PE(Protective Earth) is not used.

    Figure 4: Schematic with oscillator connected to ADE9113A XTALIN

    Calculate the PCB cap area needed for layout see Figure 6, 15.2mm * 28.7mm = 437.3 sqmm Cap area. Using FR4 dielectric K of 4.2 and thickness of 0.5mm = 32.5pf of PCB capacitance. This is the minimum and larger is better. The recommended PCB capacitance is approx. 50% of the cap recommended in the ADE7913/33 AN-1332: Architecting a Direct, 3-Phase Energy Meter with Shunts Using the ADE7912/ADE7913

    C= K*Eo*A/D, where Eo= 8.854x10-12

    where:

    K is the dielectric constant of the material,

    A is the overlapping surface area of the plates in m2,

    d is the distance between the plates in m, and C is capacitance

    https://www.daycounter.com/Calculators/Plate-Capacitor-Calculator.phtml

    Key to layer stackup and layout images

    Layer 1 Signal (Red)

    Layer 2 AGND_1 (Green) Must be under the signal layer 1 to shield layer 1 from layer 3. Layer 3 may have a potential difference of the line voltage to layer 1.

    Layer 3 DGND (Purple) this layer may have line voltage difference to layer 1. This requires layer 2 as a shield between layer 1 and layer 3.

    Layer 4 No copper under isolated side

    Figure 5: Layer 2 (AGND_1) Shields layer 1(Signal) from layer 3 (Dgnd)

    Figure 6: Capacitor is made up of the overlap of Layer 2 (AGND_1) and Layer 3(Dgnd)

    Figure 7: Layer 2 Minimize Vias

    The VIAS at the GND pins are critical and should be close as possible to the ADE9113 pins shown in Figure 8.

    Figure 8: Critical VIA Placement

    Through VIAS should be limited. The through VIA keep out on layer 3 (Dgnd) around the via takes away for the PCB cap See Figure 9.
