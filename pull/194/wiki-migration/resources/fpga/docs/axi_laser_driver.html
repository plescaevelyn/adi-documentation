    AXI Laser Driver IP core

    The axi_laser_driver IP generate all the necessary control signal for the ad_fmclidar1_ebz prototyping platform, to drive the laser, switch between the TIA channels and synchronize all the data flow inside the FPGA. This documentation covers the main functionality of the IP core and its software programmable registers.

    Features

    • PWM generator with a free running counter

    • Software configurable frequency and pulse width for the PWM

    • Control and status lines for the ADP3624 MOSFET driver

    • Software configurable interrupts

    • Software configurable TIA channel sequencer

    Interface

    Pin

    Type

    Description

    AXI_S_MM interface

    s_axi_*

    Standard AXI Slave Memory Map interface

    External clock

    ext_clk

    input

    An external clock input. If ASYNC_CLK_EN is set, this clock will be used as the core clock.

    Laser driver controls and status

    driver_en_n

    output

    This line will control the MOSFET driver’s shot down pin. If set the MOSFET driver is not working.

    driver_pulse

    output

    Generated pulse, which is used to drive the laser

    driver_otw_n

    input

    Over temperature warning flag of the MOSFET driver.

    driver_dp_reset

    output

    A reset signal, which asserts right before the generated pulse. This signal can be used for preconditioning various IPs of the data path.

    tia_chsel

    output[7:0]

    Control lines for the TIA channel multiplexer.

    output

    Interrupt signal
