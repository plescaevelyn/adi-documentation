    A2B Bus Analyzer and A2B Analyzer HP Hardware Setup

    Introduction

    The guidelines on these pages are intended to help A2B users configure and use the ADI A2B Analyzer Studio tool in Monitor or Emulator mode for A2B1.0 and A2B2.0 networks on A2B Bus Analyzer and A2B Analyzer HP respectively.

    Getting Started with A2B Analyzer HP

    A2B Analyzer HP Setup

    1. Connect the A2B Analyzer HP to your PC using the supplied USB-C to USB-C cable. You may also use USB-C to USB-A cable with 24V power supply.

    1. Connect 24V power supply to A2B Analyzer HP. A2B Analyzer HP supports both power and data over USB-C. USB-C power alone may not be sufficient to use the A2B Analyzer HP to its full capacity. When the power is not sufficient, an error message is displayed in the User Interface. To allow the connection a 24V power supply must be connected.

    1. Ensure your A2B Analyzer Studio software is installed, and upgraded as explained in A2B Analyzer Studio Quick Start Guide Section “Installation and Upgrades”.

    2. Launch A2B Analyzer Studio. Connect to your A2B Analyzer HP device by selecting it from the device selection drop-down menu then clicking the Power button. Once connected, the Power button becomes green.

    Hardware setup for using A2B Analyzer HP as Bus Monitor

    1. Complete steps in A2B Bus Analyzer and A2B Analyzer HP Hardware Setup.

    2. For custom cabling, please use the below pinout details.

    • P1 (towards main): 1 - AN, 2 - AP(left to right)

    • P2 (towards sub):   1 - BN, 2 - BP (left to right)

    1. Connect A2B Analyzer HP unit between two nodes in an A2B network. You must use personality module 2 for monitor functionality.

    Note: It is recommended to place the A2B Analyzer HP between the Main and first Sub node (Sub node0) to allow capturing all events on the bus.

    1. If ADI evaluation boards are used for main and sub-node, please refer the following path for documentation on how to set up a network using ADAA245x-EV-A1CA + ADAA24xx-EV-B2BX eval boards.

    • <Installation path of SigmaStudio+>\docs\root-qsg\resources\pages\quick-start-guide\gui\examples\adaa245x-ev-a1ca-rev-d_adaa24xx-ev-b2bx-rev-c_3-node.html

    • Also, please refer to the specific jumper settings for main node and sub-node.

    1. Follow the instructions in A2B Analyzer Studio Quick Start Guide section *”Using A2B Analyzer HP as Bus Monitor” * for using A2B Analyzer Studio software.

    Hardware setup for using A2B Analyzer HP as Node Emulator

    Main Node Emulator

    1. Complete steps in A2B Bus Analyzer and A2B Analyzer HP Hardware Setup.

    2. For custom cabling, please use the below pinout details.

    • P1 (towards main): 1 - AN, 2 - AP(left to right)

    • P2 (towards sub):   1 - BN, 2 - BP(left to right)

    1. Connect Sub node or chain of Sub nodes to the node emulator port labelled “TOWARDS SUB”. You must use personality module 1 for emulator functionality.

    1. Follow the instructions in A2B Analyzer Studio Quick Start Guide section *”Using A2B Analyzer HP as Node Emulator - Main Node Emulator” * for using A2B Analyzer Studio software.

    Sub Node Emulator

    1. Complete steps in A2B Bus Analyzer and A2B Analyzer HP Hardware Setup.

    2. For custom cabling, please use the below pinout details.

    • P1 (towards main): 1- AN, 2 - AP(left to right)

    • P2 (towards sub): 1 - BN, 2 - BP(left to right)

    1. Insert the A2B Analyzer HP unit in an A2B network at a desired Sub node position to be emulated. Connect the unit to an upstream node, using TOWARDS MAIN, and to a downstream node, using TOWARDS SUB (if it is not the last node). You must use personality module 1 for emulator functionality.

    1. Follow the instructions in A2B Analyzer Studio Quick Start Guide section *”Using A2B Analyzer HP as Node Emulator - Sub Node Emulator” * for using A2B Analyzer Studio software.

    Getting Started with A2B Bus Analyzer

    A2B Bus Analyzer Setup

    1. Connect the A2B Bus Analyzer to your PC using the supplied USB-A to Micro-B cable.

    1. Connect 12V power supply to A2B Bus Analyzer.

    1. Ensure your A2B Analyzer Studio software is installed, and upgraded as explained in A2B Analyzer Studio Quick Start Guide Section “Installation and Upgrades”.

    2. Launch A2B Analyzer Studio. Connect to your A2B Bus Analyzer device by selecting it from the device selection drop-down menu then clicking the Power button. Once connected, the Power button becomes green.

    Hardware setup for using A2B Bus Analyzer as Bus Monitor

    1. Complete steps in A2B Bus Analyzer and A2B Analyzer HP Hardware Setup.

    2. For custom cabling, please use the below pinout details.

    • P1 (towards main): 1 - NC, 2 - AN, 3 - AP, 4 – NC (left to right)

    • P2 (towards sub): 1 - NC, 2 - AP, 3 - AN, 4 – NC (left to right)

    1. Connect A2B Bus Analyzer unit between two nodes in an A2B network.

    Note: It is recommended to place the analyzer between the Main and first Sub node (Sub node0) to allow capturing of all events on the bus.

    1. Follow the instructions in A2B Analyzer Studio Quick Start Guide section *”Using A2B Analyzer HP as Bus Monitor” * for using A2B Analyzer Studio software.

    Hardware setup for using A2B Bus Analyzer as Node Emulator

    For customer cabling, please use the below pinout details:

    High power connector:

    • P7 (towards sub): 1 - GND, 2 - BP, 3 - BN, 4 – GND (left to right)

    • P8 (towards main): 1 - GND, 2 - AN, 3 - AP, 4 – GND (left to right)

    Standard power connector:

    • P6 (towards sub): 1 - BP, 2 – BN (left to right)

    • P3 (towards main): 1- AN, 2- AP (left to right)

    Main Node Emulator

    1. Complete the steps in A2B Bus Analyzer and A2B Analyzer HP Hardware Setup.

    2. Connect Sub node or chain of Sub nodes to the node emulator port labelled “TOWARDS SUB” on STD or HIGH PWR depending on the power configuration of the node being emulated

    1. Follow the instructions in A2B Analyzer Studio Quick Start Guide section *”Using A2B Bus Analyzer as Node Emulator - Main Node Emulator” * for using A2B Analyzer Studio software.

    Sub Node Emulator

    1. Complete steps in Section A2B Bus Analyzer and A2B Analyzer HP Hardware Setup.

    2. Insert the A2B Bus Analyzer unit in an A2B network at a desired Sub node position to be emulated. Connect the unit to an upstream node, using TOWARDS MAIN, and to a downstream node, using TOWARDS SUB (if it is not the last node), on STD or HIGH PWR node emulator ports depending on the power configuration of the node being emulated.

    1. Follow the instructions in A2B Analyzer Studio Quick Start Guide. section *”Using A2B Bus Analyzer as Node Emulator - Sub Node Emulator”  * for using A2B Analyzer Studio software.
