Uploading Images and Files
--------------------------

You have the ability to insert images and files into your Wiki content. To
access them from the Wiki, you must first upload them. To do this, go to the
page where you want to insert the file and click “Edit this Page”.

On the editing toolbar, click the icon labeled “Add Images and other files”.

.. image:: https://wiki.analog.com/_media/wiki/help/getting-started/wiki-add-images.png
   :width: 650
   :height: 63px

A dialog box will appear that allows you to browse for a picture or file on your
computer and upload it to the current folder by clicking upload. To upload to a
different folder on the Wiki, select it from the left menu.

.. image:: https://wiki.analog.com/_media/wiki/help/getting-started/media-files.png

Inserting Images and Files
--------------------------

Once you have uploaded your image or file, go to the area on the page you would
like it to appear and click the “Add images and other files” icon again (you can
also insert images and files right after you upload them if you place your
cursor where you want to insert the image or file prior to uploading it). Select
your image or file from the dialog box by clicking on its name. This will insert
the code to display the image or link to the file at this spot in the Wiki.

Alternatively, if you know the name of your image or file, you can also insert
it manually by using the following Wiki syntax:

`imagename.xxx <https://wiki.analog.com/_media/wiki/help/getting-started/imagename.xxx>`_
