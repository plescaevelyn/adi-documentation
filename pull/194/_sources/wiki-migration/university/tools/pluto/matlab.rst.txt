ADALM-PLUTO
===========

To use Pluto with MATLAB, you must install the `Pluto Hardware Support Package <https://www.mathworks.com/hardware-support/adalm-pluto-radio.html>`_.

This can be accomplished from within MATLAB using the addon explorer, which can
be launched as shown here:

|image1|

Next, search for ADALM-Pluto and select the desired result:

|image2|

Finally, click on the successive install options and follow the guided wizard
that will appear:

|image3|

If the installer fails, please contact `MathWorks support <https://www.mathworks.com/support/contact_us.html?s_tid=sp_ban_cs>`_.
