EVALUATING THE LTC2000/LTC2000A DIGITAL-TO-ANALOG CONVERTER USING DC2303A
=========================================================================

Preface
-------

This user guide describes both the hardware and software setup needed to acquire data capture from :adi:`DC2303A <ltc2000/#evaluation-kit>` evaluation board to characterize :adi:`ltc2000` / :adi:`ltc2000a` 16-/14-/11-bit, 2.5-/2.7-GSPS, RF Digital-to-Analog Converters.

This guide provides instructions for using DC2303A with KC705 controller board. Link to the User Guide for DC2085A with Stratix IV is provided for customers who still have this evaluation board.

Quick Start Guide
-----------------

-  Download and install the **LTDACGen**.
-  Attach DC2303A and DC2159A to KC705 through the FPGA board’s HPC and LPC FMC connectors, respectively.
-  Connect **J1** and **J2** of the DC2233A to **J3** and **J1** of the DC2303A.
-  Connect a **50-ohm resistor** to DC2159A across CN2 pins 23 and 2 then connect the board to PC using a USB cable.

.. important::

   Note: The 50-ohm resistor is required to allow the use of DC2159A for DAC evaluation.


-  Switch on KC705. If the board is not pre-programmed, follow the steps below to configure the FPGA:

   -  Download and install **Vivado**.

      -  Connect KC705 **JTAG** to PC using a USB cable.
      -  Launch Vivado then click **Open Hardware Manager**.
      -  Select **Open target > Auto Connect**. Vivado should be able to detect and connect to the KC705 (See *Hardware* panel).
      -  On the *Hardware* panel, right click on the FPGA and select **Program Device**.
      -  Click on the **[…]** button beside the *Bitstream file* field and navigate to the **bit** file inside the **LTDACGen installer** folder.
      -  After browsing to the bitstream file, click **Program** to program the FPGA.
      -  Once programming is done, right click on the FPGA target on the *Hardware* panel then select **Close Target**.
      -  Right click on the server (**localhost**) on the *Hardware* panel then select **Close Server**.
      -  Exit Vivado and disconnect the USB cable connecting the KC705 to PC.

-  Power up the DC2303A evaluation board by connecting it to a 5-Vdc, 2-A power supply through the +5V turret.
-  Connect the continuous wave generator for clock input to **J4** using an SMA cable. Set the clock input to **2.5 GHz** and **10 dBm**.
-  Connect the spectrum analyzer to **J3** of the DC2233A using an SMA cable.
-  Open **LTDACGen**. All status indicators should be green as shown in Figure 2.\ |image2|\

.. container:: centeralign

   \ *Figure 2. Launching LTDACGen (FPGA Connected)*-  Select **Sine** from the dropdown menu under the*FPGA LVDS Data* panel then press **Add**. As can be seen in Figure 3, the frequency should be set to 400 MHz by default.\ |image3|\

.. container:: centeralign

   \ *Figure 3. Generating a Sine Wave Pattern using LTDACGen*-  Change the*Center Frequency* to **200 MHz** then click the **Update FPGA** button to send the data to the FPGA and program the DAC.  The FFT plots similar to Figures 4 should appear in the signal/spectrum analyzer.

Troubleshooting
---------------

-  Check if the USB cables are directly plugged into the PC USB ports to ensure enough power is supplied to the USB interface of KC705 and DC2159A.
-  Check if a 50-ohm resistor is connected across CN2 pins 2 and 23.
-  Power cycle the KC705 controller board then perform Step 5 of the Quick Start Guide.
