Evaluation Board for the LTC6563 Four-Channel Differential-Output Transimpedance Amplifier
==========================================================================================

This is an addendum to the user guide available at :adi:`media/en/technical-documentation/user-guides/eval-ltc6563.pdf`. Please refer to that guide to get started.

Programming the EVAL-LTC6563 SDP-K1 Evaluation Board
----------------------------------------------------

If the SDP-K1 included in the EVAL-LTC6563 kit was not pre-programmed with the correct .bin file, the expected terminal window will not appear. To correct this problem, please follow the below steps to load the .bin file into the SDP-K1.

-  Unzip this `bin file <https://wiki.analog.com/_media/resources/eval/eval-ltc6563-tqfn.sdp_k1_bin.zip>`_ and add it to a directory on your computer.
-  Move the P14 jumper on the SDP-K1 board to the "3.3V" position (to the right).
-  Plug one end of the USB cable in to the computer where the .bin file is loaded.
-  Plug the other end of the USB cable in to the SDP-K1. The SDP-K1 should appear as a new drive in the Windows File Explorer menu.
-  Select the .bin file in File Explorer. Drag and drop the .bin file onto the SDP-K1 drive location in File Explorer.
-  The process should take 30 seconds from start to finish. After this, the SDP-K1 is ready to use with the EVAL-LTC6563 evaluation board.
