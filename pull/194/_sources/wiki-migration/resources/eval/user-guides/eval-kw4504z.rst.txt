Two Op amp DC Test Loop KWIK Lecture + Lab
==========================================

General Description
-------------------

The Two Op Amp DC Test Loop **K**\ now-how **W**\ ith **I**\ ntegrated **K**\ nowledge (KWIK) Lecture and Lab is a training module comprised of a descriptive lesson and a hands-on activity using the ADALM2000 and EVAL-KW4504Z, with the goal of understanding the DC parameters of an operational amplifier.

The EVAL-KW4504Z or the Op Amp DC Test Loop KWIK Demo board is a board developed
for this specific module. Together with the ADALM2000, it was designed to
educate users on how to evaluate and measure the DC parameters of an operational
amplifier, including the Input Offset Voltage (Vos), Power Supply Rejection
Ratio (PSRR), Common-Mode Rejection Ratio (CMRR), and Open-Loop Gain (Aol). It
also has the capability to measure and plot the relationship between the Input
Offset Voltage vs. Common-Mode Voltage (Vos vs. CMV), Input Offset Voltage vs.
Power Supply Voltage (Vos vs. Vs), and Input Offset Voltage vs. Output Voltage
(Vos vs. Vout).

By default, EVAL-KW4504Z uses AD8601 as its DUT (Device Under Test). AD8601 is a
single channel, rail-to-rail, input and output, single-supply amplifier
featuring a very low offset voltage and a wide signal bandwidth.

Objectives
----------

-  Educate users on evaluating and measuring the key DC parameters of an
   operational amplifier, particularly:

   -  Input Offset Voltage (Vos),

      -  Power Supply Rejection Ratio (PSRR),

         -  Common-Mode Rejection Ratio (CMRR),
         -  Open-Loop Gain (Aol)

-  Plot and show the relationship between the following:

   -  Input Offset Voltage vs. Common-Mode Voltage (Vos vs. CMV)

      -  Input Offset Voltage vs. Power Supply Voltage (Vos vs. Vs)

         -  Input Offset Voltage vs. Output Voltage (Vos vs. Vout)
