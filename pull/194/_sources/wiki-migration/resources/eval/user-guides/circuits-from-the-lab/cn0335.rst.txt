CN0335 Software User Guide
==========================

Overview
--------

:adi:`CN0335` provides robust and complete solution for ±10 voltage to digital conversion with isolation, for measurements where, voltage is used as standard interface. The design solution is optimized for high precision and low cost measurement, using only three active devices, and has a total unadjusted error of less than 0.1% FSR. The accuracy depends on the calibration. The circuit shown in Figure 1 incorporates :adi:`AD8606ARZ` OpAmp, :adi:`AD7091RBRMZ` 12-bit successive approximation (SAR) ADC, and :adi:`ADuM5401ARWZ` isolator to create a standard ±10 voltage measurement system. The circuit has 12 pin PMOD connector on board, which could be used for connection to a customer microprocessor or FPGA. This :adi:`CN0335` circuit note discusses the design steps needed to optimize the circuit shown in Figure 1 for a specific bandwidth including noise analysis and component selection considerations.

The performance of the circuit could be demonstrated with the use of Analog Devices SDP controller :adi:`EVAL-SDP-CB1Z` and SDP-to-PMOD Interposer Board :adi:`SDP-PMD-IB1Z`, both optional purchase items. This user guide will discuss how to use the evaluation software to collect data from the EVAL-CN0335-PMDZ Evaluation Board :adi:`CN0335`

   :width: 900px

*Figure 1. ±10 V Single Supply Analog to Digital Conversion with Isolation (All connections and Decoupling Not Shown)*

Required Equipment
------------------

-  :adi:`EVAL-SDP-CB1Z` Controller Board (**SDP-B Board**)
-  EVAL-CN0335-PMDZ Evaluation Board (**CN-0335 Board**)
-  :adi:`EVAL-CFTL-6V-PWRZ` **(+6V Power Supply)** or equivalent
-  :adi:`SDP-PMD-IB1Z` SDP-to-PMOD interposer board
-  Voltage Source (Voltage Generator)
-  Multimeter(to measure the input current)
-  `CN0335 Evaluation Software <https://wiki.analog.com/ftp/ftp.analog.com/pub/cftl/cn0335>`_

   -  (supplied with provided CD in kit)

-  PC with the following *Minimum Requirements*

   -  Windows XP Service Pack 2 (32-bit)
   -  USB type A Port
   -  Processor rated at 1GHz or faster
   -  512 MB RAM and 500 MB available hard disk space

-  USB type A to USB type mini-B cable

General Setup
-------------

-  The :adi:`EVAL-CFTL-6V-PWRZ` **(+6V DC Power Supply)** powers the :adi:`EVAL-CN0335-SDPZ <CN0335>` **(CN335 Board)** via the DC barrel jack.
-  The :adi:`SDP-PMD-IB1Z` **(Interposer Board)** connects to the :adi:`EVAL-SDP-CB1Z` **(SDP-B Board)** via the 120-Pin connector A.
-  The :adi:`EVAL-SDP-CB1Z` **(SDP-B Board)** connects to the PC via the USB cable.
-  The :adi:`EVAL-CN0335-SDPZ <CN0335>`\ **(CN0335)** connects to the :adi:`EVAL-PMD-IB1Z` **(Interposer board)** via the 12-Pin header PMOD connector **(J1 and J3)**
-  The ±10V source **(Voltage Generator)** connects to the :adi:`EVAL-CN0335-SDPZ <CN0335>` **(CN0335 Board)** via the terminal block **J2**

::

   -**Connect/Reconnect Button**
   * When this button is pressed, the SDP-B Board makes a USB connection to the CN0335 Board.  A connection to the SDP-B Board must be made to use the software.
   -**Run Button**
   * When this button is pressed, the SDP-B Board will collect conversion data and present the acquisitions in the chart.

   * When this button is pressed, the software stop collecting data from the CN0335 Board

   * When this button is pressed, the software will clear all data collected from the chart history.

   * When this button is pressed, the software will save the data collected to a tab delimited ASCII spreadsheet file.

   * __Acquire DataMain__ - Clicking this tab brings the data collection graph to the front.

   -**Numerical Indicator**
   * This indicator displays the current reading value. The value units depend on the chosen graph units (10).

   * These controls allow the user to zoon in, zoom out, and pan through the data collected.

   * This control determines the units of the data displayed in the chart and the numerical indicator.

Establishing a USB Connection Link
~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~

-  Follow the instructions to properly install the software and connect the hardware as described in the previous sections.
-  Open the file named **CN0335.exe** in the installation directory.

.. note::

   \ **NOTE:** If the software was installed to the default location it will be found at

   **C:\\Program Files\\Analog Devices\\CN0335\\CN0335.exe**\


- Click the **Connect/Reconnect Button**. A window with a progress bar will load.



|image17|

   \* Upon success, the **System Status String Indicator** will display *SDP Board Ready to Acquire Data*

Calibrating the CN0335 Board
~~~~~~~~~~~~~~~~~~~~~~~~~~~~

-  **Establish a USB Connection Link**.
-  Click the **Calibrate System** tab.
-  Apply first calibration current to the input connector **J2** of **CN0335**.
-  Write down the applied current to **Calibration Current 1** indicator.
-  Press **Calibrate 1** button.
-  Apply second calibration current to the input connector **J2** of **CN0335** and different from the previous calibration current.
-  Write down the applied current to **Calibration Current 2** indicator.
-  Press **Calibrate 2** button.

Capturing a Data
~~~~~~~~~~~~~~~~

-  **Establish a USB Connection Link**.
-  Click the **Run Button**
-  Click the **Stop Button** when acquisition is complete.

Saving Data to a Spreadsheet File
~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~

-  **Establish a USB Connection Link**.
-  **Capture Data**.
-  Click the **Save Data Button**.

|image18|

   \* Click the **OK Button**.
-  Browse to the directory location where the spreadsheet file is to be saved.
-  Name the file.
-  Click the **OK Button**.
-  Open with Notepad or similar. The saved data shows the ADC Codes (decimal)



.. note::

   | The software saves the spreadsheet file as ASCII text with columns separated by tabs.
