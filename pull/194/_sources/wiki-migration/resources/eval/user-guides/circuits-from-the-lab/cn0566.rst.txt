ADALM-PHASER (CN0566) Phased Array Exploration Platform
=======================================================

Introduction
------------

Phased array communications and radar systems are finding increased use in a variety of applications. This places a greater importance on training engineers and rapidly prototyping new phased array concepts. However, both those imperatives have historically been difficult and expensive. That is why Analog Devices launched the ADALM-PHASER. It is a low cost, simplified phased array radar which allows real beamforming hardware to be used for education, project proposals, and even software development. Most of the labs are run on a Python GUI, with several helper scripts. These can be run on a remote host computer, or directly on the phaser's Raspberry Pi computer with the addition of a keyboard, mouse, and display. For more advanced application development, the Phaser is also supported in the MATLAB RF Microwave Toolbox. Equivalent MATLAB lab exercises are in development, and will be added below as they are completed.
