HT DAQ Viewer Application
-------------------------

The following zip file contains the executable for the EV-HT-200CDAQ1 HT Data Viewer application. There is no installer for this project. Simply unzip the archive in the directory in which you wish to store it and double click to run it. MathNet.Numberics.dll must always be located in the same directory. For more information on installing the HT DAQ Viewer application, see the :doc:`user guide </wiki-migration/resources/eval/user-guides/high-temp/ev-ht-200cdaq1/usersguide>`

Click here to download the HT DAQ Viewer application: `ev-ht-200cdaq1_ht_daq_viewer_1p0.zip <https://wiki.analog.com/_media/resources/eval/user-guides/high-temp/ev-ht-200cdaq1/ev-ht-200cdaq1_ht_daq_viewer_1p0.zip>`_
