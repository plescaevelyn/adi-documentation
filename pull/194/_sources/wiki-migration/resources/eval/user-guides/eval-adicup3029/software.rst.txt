Software Pack and Board Support Packages
========================================

The ADICUP3029 platform is built upon a software framework that has many layers.
In order to run any of the examples shown with this platform, it is extremely
important that all users have the following software packs installed in their
tools.

-  :doc:`ADuCM302x Pack </wiki-migration/resources/eval/user-guides/eval-adicup3029/software/aducm302x>`
-  :doc:`ADICUP3029 Pack </wiki-migration/resources/eval/user-guides/eval-adicup3029/software/adicup3029>`
-  :doc:`Sensor Pack </wiki-migration/resources/eval/user-guides/eval-adicup3029/software/sensors>`
-  :doc:`BLE Software Pack </wiki-migration/resources/eval/user-guides/eval-adicup3029/software/ble_sw>`
-  :doc:`Wi-Fi Software Pack </wiki-migration/resources/eval/user-guides/eval-adicup3029/software/wifi_sw>`

The simplified diagram below describes the interaction between all packs and the
applications, and how the packs are built on top of each other. In order to
import, create, or debug applications with the ADICUP3029, it is important to
ensure all packs are installed before running an example.
