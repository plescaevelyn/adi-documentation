Tools and Driver Details
========================

This chapter provides all the necessary steps to download, install, and setup
the software environment from Analog Devices. There is also information on the
different USB modes and drivers needed.

It contains four main sections:

-  :doc:`CrossCore Download & Installation Guide </wiki-migration/resources/eval/user-guides/eval-adicup3029/tools/cces_setup_guide>` - Provides all the necessary instructions on how to download and install the CrossCore Embedded Studios software development environment.
-  :doc:`CrossCore Quickstart User Guide </wiki-migration/resources/eval/user-guides/eval-adicup3029/tools/cces_user_guide>` - Provides information about using the CrossCore Embedded Studios IDE, in particular, the process of importing, building, debugging, and creating user applications for the EVAL-ADICUP3029.
-  :doc:`Hardware Drivers and USB Storage Drives </wiki-migration/resources/eval/user-guides/eval-adicup3029/tools/adicup3029_hw_drivers>` - Provides detailed information how to load pre-compiled .HEX or .BIN files using the USB drive of the EVAL-ADICUP3029 board.
-  :doc:`Using EVAL-ADICUP3029 with IAR and Keil IDEs </wiki-migration/resources/eval/user-guides/eval-adicup3029/tools/keil_iar_support>` - Provides detailed information how to use the EVAL-ADICUP3029 board with IAR Embedded Workbench and Keil µVision.
