How to run the AD9361 Receive Testbench?
========================================

-  Open the Simulink Library browser.
-  In the library browser, find RF Blockset Models for Analog Devices Transcievers.
-  Under AD9361 Testbenches, find ad9361_RX and double click it to open the
   testbench.

.. image:: https://wiki.analog.com/_media/university/tools/rfblkset_mdls_run_testbench/lib_browser.png
   :align: center
   :width: 600

-  Double-click Manual Switch block to toggle between continuous wave (CW) or
   LTE source input.

.. image:: https://wiki.analog.com/_media/university/tools/rfblkset_mdls_run_testbench/manual_switch.png
   :align: center
   :width: 600

-  Set the Simulation Stop Time to 0.005 s.

.. image:: https://wiki.analog.com/_media/university/tools/rfblkset_mdls_run_testbench/simulation_time.png
   :align: center
   :width: 600

-  Run the simulation using the Play button.
