Installing RF Blockset Models for Analog Devices RF Transceivers
================================================================

MathWorks' RF Blockset Models of Analog Devices RF Agile Transceivers AD9361 and
AD9371 can be installed from within MATLAB using the add-on explorer, as shown
below:

.. image:: https://wiki.analog.com/_media/university/tools/adi_mw_rfblks/install0.png
   :width: 800

Next, search for RF Blockset Models for Analog Devices RF Transceivers and
select it:

.. image:: https://wiki.analog.com/_media/university/tools/adi_mw_rfblks/install1.png
   :width: 800

Finally, click on the install button and follow the instructions:

.. image:: https://wiki.analog.com/_media/university/tools/adi_mw_rfblks/install2.png
   :width: 800

If the installer fails, please contact `MathWorks support <https://www.mathworks.com/support/contact_us.html?s_tid=sp_ban_cs>`_.
