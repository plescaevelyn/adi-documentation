Application Demos
=================

The ADICUP360 is designed with the idea of flexibility, scalability, and rapid
prototyping in mind. Meaning that you can stack more than one Arduino shield or
attach more than one PMOD hardware module to this development platform at a
single time. And by combining the software/hardware projects from the examples
shown, you can create more complex systems and subsystems quickly. Here is an
example of some more complex examples that create a more complete and closed
loop system/solution.
