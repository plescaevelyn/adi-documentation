Outline
-------

For those that don't want to load up the Allegro viewer, here is a basic outline/component placements of the board. The PCB measures 100mm x 62mm and is the same form factor as the RFSOM

Size
----

The size of the board (not including the SMA connectors, which project beyond the edge of the board) is 110mm x 69mm. The mounting holes align with the carrier card (EK-Z7-ZC706-G), and are shown below.
