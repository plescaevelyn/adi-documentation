Help & Support
--------------

.. hint::

   
   -  The :doc:`carriers (abbrevations can be found here) </wiki-migration/resources/fpga/docs/hdl>` are commonly available FPGA evaluation boards.
   -  The :doc:`HDL user guide </wiki-migration/resources/fpga/docs/hdl>` contains all the documentation, build instructions and register map tables.
   -  The following quick links allows you to browse the github repository for a list of current `branches <http://github.com/analogdevicesinc/hdl/branches/active>`_, `library components <http://github.com/analogdevicesinc/hdl/tree/master/library>`_, and `projects <http://github.com/analogdevicesinc/hdl/tree/master/projects>`_.
   -  Questions? We can help with :ez:`FPGA questions <community/fpga>`, :ez:`Linux driver questions <community/linux-device-drivers/linux-software-drivers>`, :ez:`No-OS Drivers <community/linux-device-drivers/microcontroller-no-os-driver questions>`.
