Avalon Transceiver Configuration Core (avl_adxcfg)
==================================================

The Avalon Transceiver Configuration (avl_adxcfg) core is a supplemental core to the Avalon Transceiver (avl_adxcvr) core. It implements the avalon slave merge function that is required to allow transceiver sharing across multiple 'avl_adxcvr' cores.
