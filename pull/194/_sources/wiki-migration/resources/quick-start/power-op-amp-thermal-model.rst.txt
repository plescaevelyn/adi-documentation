LTspice User Guide – Power Op Amp Thermal Model
===============================================

Overview
========

The Power Op Amp Thermal Model (AD8460, ADHV4710) is a Second Order, Steady State, Thermal Model for use with the AD8460 and ADHV4710 Power Op Amp LTspice Electrical Models. The model will enable the user to determine his worst-case power dissipation and design for the appropriate heatsink, given a desired maximum junction temperature for the targeted end application. Ensure you are running the latest version of LTspice which can be found at: :adi:`LTspice Info center <en/design-center/design-tools-and-calculators/ltspice-simulator.html>`
