Development and Debug Commands
==============================

The purpose of these commands is to help users or developers of the Audio Starter debug system level issues, issues with hardware connections, etc. This section describes each of the development and debug related shell commands in detail.

--------------

Automated On-Target Command Execution (run)
-------------------------------------------

.. note::

   Short options support status:

   
   -  ADZS-SC589-MINI - v4.0.0 or greater
   -  ADZS-SC584-EZLITE - TBD
   -  EV-SC594-SOM - TBD
   -  EV-SC598-SOM - TBD
   


+-------------------+---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------+
| **Description**   | This command executes a series of shell commands that are stored in a *.cmd* that lives in one of the available file-systems on the Audio Starter. This can be useful for quickly and automatically running a series of commands in a sequence, especially as the state of the Audio Starter is not persistent across a power-cycle.                                            |
|                   | For example, if a user is using the Audio Starter for a demo application that needs to set up a series of routes with a wav file enabled and streaming, it may be more convenient to simply save that series of commands to a file on the Audio Starter, rather than re-type them into the shell after each power cycle. Generally these type of files are denoted as *\*.cmd*. |
|                   | Outside of this command, *.cmd* files can also be run via the pushbuttons (*pushbtn1.cmd* and *pushbtn2.cmd* in the default file-system) on the hardware and automatically at startup (file must be named *shell.cmd* and must reside in *sf:*).                                                                                                                                |
|                   | To run a command file that resides in the non-default file-system, append the device drive identifier to the filename. See the command *drive* for more details.                                                                                                                                                                                                                |
|                   | Run command may be interrupted by pressing **CTRL + C**.                                                                                                                                                                                                                                                                                                                        |
+-------------------+---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------+
| \*\* Syntax \*\*  | *run [options] <file1> [<file2> ...]*                                                                                                                                                                                                                                                                                                                                           |
|                   | |image25|                                                                                                                                                                                                                                                                                                                                                                       |
+-------------------+---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------+
| **Inputs**        | **file1** - This is the name of the *.cmd* file to execute.                                                                                                                                                                                                                                                                                                                     |
|                   | *Range(string, required): any*                                                                                                                                                                                                                                                                                                                                                  |
|                   | *Note that subsequent files are optional.*                                                                                                                                                                                                                                                                                                                                      |
|                   | **OPTIONS**                                                                                                                                                                                                                                                                                                                                                                     |
|                   | **-l <loop count>**                                                                                                                                                                                                                                                                                                                                                             |
|                   | This option can be used to run one or more command files the determined number of times. If not set, command files will be run once.                                                                                                                                                                                                                                            |
+-------------------+---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------+
| **Outputs**       | Prints any outputs executed by commands that reside in the *.cmd* file.                                                                                                                                                                                                                                                                                                         |
+-------------------+---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------+
| **Example Usage** | *run test.cmd*                                                                                                                                                                                                                                                                                                                                                                  |
|                   | This will run a file called *test.cmd* that resides within the default file-system. The commands that live within this file are visible with the *cat* command (see below). Each command is separated by a new line.                                                                                                                                                            |
|                   | |image26|                                                                                                                                                                                                                                                                                                                                                                       |
|                   | *run -l 10 file1.cmd file2.cmd*                                                                                                                                                                                                                                                                                                                                                 |

Execute Raw I2C Transfer (i2c)
------------------------------

+-------------------+---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------+
| **Description**   | This command executes a raw I2C read or write transfer. This is useful for communicating directly with on-board I2C devices for debugging purposes (without the need for modifying direct source code). Note that this command can handle both 8-bit and 16-bit register addresses (*mem_addr* in the input list below), but only supports 7-bit addressing (10-bit addressing is NOT supported). |
|                   | At the lowest layer, this command calls a single *writeRead* API, which inserts a repeated start condition in-between the write and read transaction. For example, the below scenario shows a single read, with a 16-bit register address, reading two bytes of data:                                                                                                                             |
|                   | |image52|                                                                                                                                                                                                                                                                                                                                                                                         |
|                   | The following formats are supported:                                                                                                                                                                                                                                                                                                                                                              |
|                   | **Single read with 8-bit or 16-bit memory addresses**:                                                                                                                                                                                                                                                                                                                                            |
|                   | |image53|                                                                                                                                                                                                                                                                                                                                                                                         |
|                   | |image54|                                                                                                                                                                                                                                                                                                                                                                                         |
|                   | **Burst read with 8-bit or 16-bit memory addresses**:                                                                                                                                                                                                                                                                                                                                             |
|                   | |image55|                                                                                                                                                                                                                                                                                                                                                                                         |
|                   | |image56|                                                                                                                                                                                                                                                                                                                                                                                         |
|                   | **Single write with 8-bit or 16-bit addresses**:                                                                                                                                                                                                                                                                                                                                                  |
|                   | |image57|                                                                                                                                                                                                                                                                                                                                                                                         |
|                   | |image58|                                                                                                                                                                                                                                                                                                                                                                                         |
|                   | **Burst write with 8-bit or 16-bit addresses**:                                                                                                                                                                                                                                                                                                                                                   |
|                   | |image59|                                                                                                                                                                                                                                                                                                                                                                                         |
|                   | |image60|                                                                                                                                                                                                                                                                                                                                                                                         |
|                   | *Note that the structure of the I2C read or write can vary from device to device. It is important to refer to the device's datasheet to understand the proper formatting of the I2C message. And although this command attempts to be flexible enough to cover a range of formats, it is not guaranteed to work with every single I2C device format that exists.*                                 |
+-------------------+---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------+
| \*\* Syntax \*\*  | *i2c <i2c_port> <i2c_addr> <mem_addr> <wdata> <length> [addr_len]*                                                                                                                                                                                                                                                                                                                                |
|                   | |image61|                                                                                                                                                                                                                                                                                                                                                                                         |
+-------------------+---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------+
| **Inputs**        | **i2c_port** - This is the physical I2C Port that the I2C device is connected to. Refer to the hardware schematic to understand the proper port to use.                                                                                                                                                                                                                                           |
|                   | *Range(integer, required) = [0, 5]*                                                                                                                                                                                                                                                                                                                                                               |
|                   | **i2c_addr** - This is the physical I2C address of the I2C device.                                                                                                                                                                                                                                                                                                                                |
|                   | *Range(integer, hexadecimal, required): Any 7-bit address*                                                                                                                                                                                                                                                                                                                                        |
|                   | **mem_addr** - This is the starting address in the device's register address space for reading or writing to.                                                                                                                                                                                                                                                                                     |
|                   | *Range(integer, required): [0, MAX_U16]*                                                                                                                                                                                                                                                                                                                                                          |
|                   | **wdata** - This is the raw data that will be used to write to the I2C device, in comma-separated, string format.                                                                                                                                                                                                                                                                                 |
|                   | *Range(string, comma-separated, required): Integer or hexadecimal values enclosed by double-quotes. Use empty quotes for a read-only operation.*                                                                                                                                                                                                                                                  |
|                   | **length** - The number of bytes to read.                                                                                                                                                                                                                                                                                                                                                         |
|                   | *Range(integer, required): [0, MAX_UL]*                                                                                                                                                                                                                                                                                                                                                           |
|                   | **addr_len** - The size, in bytes, of the memory address.                                                                                                                                                                                                                                                                                                                                         |
|                   | *Range(integer, optional, default = 1): [1, 2], 1 = 1-byte memory address, 2 = 2-byte memory address*                                                                                                                                                                                                                                                                                             |
|                   | **The command will try to infer this parameter value by checking the size of the i2c_addr value. If this value is greater than 255, the default memory address size will be 2-bytes rather than 1-byte. For devices which require an upper and lower byte, even in the range less than 255, the user MUST still input 2 for this parameter.**                                                     |
+-------------------+---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------+
| **Outputs**       | **On Reads**                                                                                                                                                                                                                                                                                                                                                                                      |
|                   | **I2C Device** - This is the 7-bit address that was read from                                                                                                                                                                                                                                                                                                                                     |
|                   | **addr** - This is the starting address that was read from                                                                                                                                                                                                                                                                                                                                        |
|                   | **bytes read** - This is the number of bytes that was read, in both decimal and hexadecimal format                                                                                                                                                                                                                                                                                                |
|                   | **n-byte address length** - This is the address length format that was used during the read operation                                                                                                                                                                                                                                                                                             |
|                   | **<addr>: <data>** - This is the actual data read                                                                                                                                                                                                                                                                                                                                                 |
+-------------------+---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------+
| **Example Usage** | *i2c 0 0x38 0x4000 "" 1 2*                                                                                                                                                                                                                                                                                                                                                                        |
|                   | This reads one byte from register address 0x4000 from device 0x38 (ADAU1761)                                                                                                                                                                                                                                                                                                                      |
|                   | *i2c 0 0x38 0x4002 "" 6 2*                                                                                                                                                                                                                                                                                                                                                                        |
|                   | This reads 6 bytes starting from register address 0x4002 from device 0x38 (ADAU1761)                                                                                                                                                                                                                                                                                                              |
|                   | |image62|                                                                                                                                                                                                                                                                                                                                                                                         |
|                   | |image63|                                                                                                                                                                                                                                                                                                                                                                                         |
|                   | *i2c 0 0x38 0x4000 "0x03" 0 2*                                                                                                                                                                                                                                                                                                                                                                    |
|                   | and                                                                                                                                                                                                                                                                                                                                                                                               |
|                   | *i2c 0 0x38 0x4000 "" 1 2*                                                                                                                                                                                                                                                                                                                                                                        |
|                   | This writes one byte of data (0x03) to register 0x4000 to device 0x38 and then proceeds to read the same register to verify the same contents were read back.                                                                                                                                                                                                                                     |
|                   | |image64|                                                                                                                                                                                                                                                                                                                                                                                         |

CPU System Utilization (cpu)
----------------------------

+-------------------+------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------+
| **Description**   | This command reports the processor utilization.                                                                                                                                                                                                                                                                                                                                                      |
|                   | On the ARM, this value is a percentage, whose value is inversely proportional to the amount of time spent idling. That is, the higher the percentage, the less time the processor is spent idling, indicating that more time is being spent executing in ISR context. A higher value or values approaching 100% indicates that the system is running out of cycles to execute all processes on time. |
|                   | On the SHARC processors, this value is reported in *CYCLES* and is the measurement of cycles spent processing SHARC input and output audio.                                                                                                                                                                                                                                                          |
|                   | *Note that the peak usage value reported for the ARM is cleared once this command is called. That is, the next peak value reported is the peak value calculated since the last time this command was called and is not the value reported of all time.*                                                                                                                                              |
+-------------------+------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------+
| \*\* Syntax \*\*  | *cpu*                                                                                                                                                                                                                                                                                                                                                                                                |
|                   | |image75|                                                                                                                                                                                                                                                                                                                                                                                            |
+-------------------+------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------+
| **Inputs**        | None                                                                                                                                                                                                                                                                                                                                                                                                 |
+-------------------+------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------+
| **Outputs**       | Displays cpu usage of all cores with current and peak usage                                                                                                                                                                                                                                                                                                                                          |
+-------------------+------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------+
| **Example Usage** | *cpu*                                                                                                                                                                                                                                                                                                                                                                                                |

Dynamic Memory Allocation Usage (meminfo)
-----------------------------------------

+-------------------+-----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------+
| **Description**   | This command displays the usage of all of the dynamically allocated memory in the system. For heaps managed by `umm_malloc <https://github.com/rhempel/umm_malloc>`_, each heap type, the name of the heap, the total heap size, the total amount of heap allocated and the remaining amount of heap left (units are in bytes) is displayed.  |
|                   | Additionally, for heaps managed by *umm_malloc*, the integrity of the memory is also verified. This can be a useful diagnostic when checking for buffer overflows and other areas of memory corruption during development.                                                                                                                    |
|                   | *Note that heap management is split between system (*umm_malloc*), FreeRTOS (*heap_4*) and SAE (*internally managed*).*                                                                                                                                                                                                                       |
+-------------------+-----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------+
| \*\* Syntax \*\*  | *meminfo*                                                                                                                                                                                                                                                                                                                                     |
|                   | |image91|                                                                                                                                                                                                                                                                                                                                     |
+-------------------+-----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------+
| **Inputs**        | None                                                                                                                                                                                                                                                                                                                                          |
+-------------------+-----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------+
| **Outputs**       | \**Heap <heap_name> Info:**                                                                                                                                                                                                                                                                                                                   |
|                   | **Total** = Total assigned in memory, **Allocated** = Memory/entry currently in-use, **Free** = Memory/entry no longer allocated                                                                                                                                                                                                              |
|                   | **Entries** - This refers to the number of buffers that were allocated for this heap.                                                                                                                                                                                                                                                         |

|                   | **Free**:                                                                                                                                                                                                                                                                                                                                     |
|                   | **Blocks** - This is the number of blocks remaining in the FreeRTOS heap.                                                                                                                                                                                                                                                                     |
|                   | **Current** - This is the available heap space, in bytes, remaining.                                                                                                                                                                                                                                                                          |
|                   | **Min** - This was the minimum number of bytes that were ever free.                                                                                                                                                                                                                                                                           |
|                   | **SHARC Audio Engine Info:**                                                                                                                                                                                                                                                                                                                  |
|                   | **Blocks:                                                                                                                                                                                                                                                                                                                                     |

Shell Output Redirect (shell)
-----------------------------

.. note::

   Support status:

   
   -  ADZS-SC589-MINI - v4.0.0 or greater
   -  ADZS-SC584-EZLITE - TBD
   -  EV-SC594-SOM - TBD
   -  EV-SC598-SOM - TBD
   


+-------------------+-----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------+
| **Description**   | This command provides the ability to redirect print statements from the system and shell commands to something other than standard output, i.e. to the system log or to a specified output file.                                                                                                                                                                                                                                                                                                                                                                                                                                                                |
+-------------------+-----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------+
| \*\* Syntax \*\*  | *shell [redirect] [value] [arg]*                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                |
|                   | |image114|                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                      |
+-------------------+-----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------+
| **Inputs**        | **redirect** - This is the action that the shell command should execute. Note that this may be expanded in the future.                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                          |
|                   | *Range(string, optional): [redirect]*                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                           |
|                   | **value** - This is the value associated with the where system printing should route to A value of 0 selects *none* which will completely suppresses any print statements. A value of 1 will select *stdout* which will print directly to standard out (i.e. shell, telnet, etc.). A value of 2 selects *syslog* which re-routes system printing to the system log. See :doc:`syslog </wiki-migration/resources/tools-software/sharc-audio-module/advanced-audio-projects/shell-commands/development-commands>` for more details about system logging. A value of 3 selects *file* which re-routes system printing to a file, specified by the user.            |
|                   | *Range(integer, optional, default = 1 (stdout)): [0, 1, 2, 3]*                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                  |
|                   | **arg** - This input value is ONLY used when the redirect value is set to *3:file*. Use this parameter to set the filename of your choosing for the printed output.                                                                                                                                                                                                                                                                                                                                                                                                                                                                                             |
|                   | *Range(string, required (when *value = 3*): any*                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                |
+-------------------+-----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------+
| **Outputs**       | None                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                            |
+-------------------+-----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------+
| **Example Usage** | *shell redirect*                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                |
|                   | or                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                              |
|                   | *shell redirect 1*                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                              |
|                   | This sets the system printing to standard out.                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                  |
|                   | |image115|                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                      |
|                   | *shell redirect 0*                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                              |
|                   | This completely suppresses system printing.                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                     |
|                   | |image116|                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                      |
|                   | *shell redirect 2*                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                              |
|                   | This re-routes system printing to the system log.                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                               |
|                   | |image117|                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                      |
|                   | *shell redirect 3 printFile.txt*                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                |
|                   | This re-routes system printing to a requested file.                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                             |
