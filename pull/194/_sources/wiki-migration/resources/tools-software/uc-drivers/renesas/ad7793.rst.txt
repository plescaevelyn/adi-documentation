AD7793 - Microcontroller No-OS Driver
=====================================

Supported Devices
-----------------

-  :adi:`AD7793`

Evaluation Boards
-----------------

-  :adi:`EVAL-AD7793EBZ`

Overview
--------

The :adi:`AD7793` is a low power, low noise, complete analog front end for high precision measurement applications. The AD7793 contains a low noise 24-bit Σ-Δ ADC with three differential analog inputs. The on-chip, low noise instrumentation amplifier means that signals of small amplitude can be interfaced directly to the ADC. With a gain setting of 64, the rms noise is 40 nV when the update rate equals 4.17 Hz.

.. image:: https://wiki.analog.com/_media/resources/tools-software/uc-drivers/renesas/eval_ad7793ebz.jpg
   :align: center

The goal of this project (Microcontroller No-OS) is to be able to provide
reference projects for lower end processors, which can't run Linux, or aren't
running a specific operating system, to help those customers using
microcontrollers with ADI parts. Here you can find a generic driver which can be
used as a base for any microcontroller platform and also specific drivers for
Renesas platforms.

**HW Platform(s):**

-  `Renesas Demo Kit for RL78G13 (Renesas) <https://www.renesas.com/us/en/products/microcontrollers-microprocessors/rl78-low-power-8-16-bit-mcus/yrdkrl78g13-yrdkrl78g13-demonstration-kit-rl78g13>`_
-  `Renesas Demo Kit for RX62N (Renesas) <https://www.renesas.com/us/en/products/microcontrollers-microprocessors/rx-32-bit-performance-efficiency-mcus/yrdkrx62n-yrdkrx62n-demonstration-kit-rx62n>`_

Driver Description
------------------

The driver contains two parts:

-  The driver for the AD7793 part, which may be used, without modifications, with any microcontroller.
-  The Communication Driver, where the specific communication functions for the
   desired type of processor and communication protocol have to be implemented.
   This driver implements the communication with the device and hides the actual
   details of the communication protocol to the ADI driver.

The Communication Driver has a standard interface, so the AD7793 driver can be
used exactly as it is provided. There are three functions which are called by
the AD7793 driver:

-  SPI_Init() – initializes the communication peripheral.
-  SPI_Write() – writes data to the device.
-  SPI_Read() – reads data from the device.

.. image:: https://wiki.analog.com/_media/resources/tools-software/uc-drivers/renesas/spi_architecture.png
   :align: center

.. container:: centeralign

   Driver architecture

The implementation of these three functions depends on the used microcontroller.

The following functions are implemented in this version of AD7793 driver:

+----------------------------------------------------------------------------------------------------+-------------------------------------------------------------+

+----------------------------------------------------------------------------------------------------+-------------------------------------------------------------+
| void AD7793_SetMode(unsigned long mode)                                                            | Sets the operating mode of AD7793.                          |
+----------------------------------------------------------------------------------------------------+-------------------------------------------------------------+

+----------------------------------------------------------------------------------------------------+-------------------------------------------------------------+

Renesas RL78G13 Quick Start Guide
=================================

This section contains a description of the steps required to run the AD7793
demonstration project on a Renesas RL78G13 platform.

Required Hardware
-----------------

-  `Renesas Demo Kit for RL78G13 (Renesas) <https://www.renesas.com/us/en/products/microcontrollers-microprocessors/rl78-low-power-8-16-bit-mcus/yrdkrl78g13-yrdkrl78g13-demonstration-kit-rl78g13>`_
-  :adi:`EVAL-AD7793EBZ`

Required Software
-----------------

-  `IAR Embedded Workbench for Renesas RL78 Kickstart <http://www.iar.com/en/Products/IAR-Embedded-Workbench/Renesas-RL78/>`_
-  `Applilet4 for RL78G13 <https://www.renesas.com/us/en/software-tool/ap4-applilet>`_

Hardware Setup
--------------

An EVAL-AD7793EBZ has to be interfaced with the Renesas Demonstration Kit (RDK)
for RL78G13:

::

     EVAL-AD7793EBZ J2 connector connector CS   → YRDKRL78G13 J11 connector Pin 1
     EVAL-AD7793EBZ J2 connector connector DIN  → YRDKRL78G13 J11 connector Pin 2
     EVAL-AD7793EBZ J2 connector connector DOUT → YRDKRL78G13 J11 connector Pin 3
     EVAL-AD7793EBZ J2 connector connector SCLK → YRDKRL78G13 J11 connector Pin 4
     EVAL-AD7793EBZ J3 connector connector AGND → YRDKRL78G13 J11 connector Pin 5
     EVAL-AD7793EBZ J3 connector connector AVDD → YRDKRL78G13 J11 connector Pin 6

.. image:: https://wiki.analog.com/_media/resources/tools-software/uc-drivers/renesas/eval_ad7793ebz_rl78g13.jpg
   :align: center

Software Setup
--------------

With the **Applilet3 for RL78G13** tool the following peripherals have to be configured:

CSI10 (Clocked Serial Interface 10) – For the AD7793 part and the ST7579 LCD
~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~

Choose to generate the Transmit/receive function for the CSI10 and configure the
interface with the following settings:

-  Transfer mode setting: **Single transfer mode**
-  Data length setting : **8 bits**
-  Transfer direction setting: **MSB**
-  Specification of data timing: **Type 1**
-  Transfer rate setting – Clock mode: **Internal clock (master)**
-  Transfer rate setting – Baudrate: **1000000** (bps)
-  Interrupt setting – Transfer interrupt priority (INTCSI10): **Low**
-  Uncheck the callback functions.

TM00 (Timer 00) – For the DelayMs() function
~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~

Configure TM00 as an interval timer:

-  Interval timer setting - Interval value(16 bits): **1** ms
-  Interval timer setting - Uncheck *Generates INTM00 when counting is started*
-  Interrupt setting - Uncheck // End of timer channel 0 count, generate an
   interrupt (INTM00)//

Watchdog Timer
~~~~~~~~~~~~~~

Disable the watchdog timer:

-  Choose for the Watchdog timer operation setting: **Unused** option.

Reference Project Overview
--------------------------

The reference project initializes the device, calibrates it and performs single
and continuous reads of an analog signal. The results are displayed on the LCD.

Renesas RX62N Quick Start Guide
===============================

This section contains a description of the steps required to run the AD7793
demonstration project on a Renesas RX62N platform.

Required Hardware
-----------------

-  `Renesas Demo Kit for RX62N (Renesas) <https://www.renesas.com/us/en/products/microcontrollers-microprocessors/rx-32-bit-performance-efficiency-mcus/yrdkrx62n-yrdkrx62n-demonstration-kit-rx62n>`_
-  :adi:`EVAL-AD7793EBZ (ADI) <EVAL-AD7793EBZ>`

Required Software
-----------------

-  `High-performance Embedded Workshop for RX62N family <https://www.renesas.com/us/en/software-tool/high-performance-embedded-workshop>`_

Hardware Setup
--------------

An EVAL-AD7793EBZ board has to be interfaced with the Renesas Demonstration Kit
(RDK) for RX62N:

::

     EVAL-AD7793EBZ J2 connector Pin DOUT → YRDKRX62N J8 connector Pin 22
     EVAL-AD7793EBZ J2 connector Pin DIN  → YRDKRX62N J8 connector Pin 19
     EVAL-AD7793EBZ J2 connector Pin SCLK → YRDKRX62N J8 connector Pin 20
     EVAL-AD7793EBZ J2 connector Pin CS   → YRDKRX62N J8 connector Pin 15
     EVAL-AD7793EBZ J3 connector Pin AVDD → YRDKRX62N J8 connector Pin 3
     EVAL-AD7793EBZ J3 connector Pin DGND → YRDKRX62N J8 connector Pin 4

.. image:: https://wiki.analog.com/_media/resources/tools-software/uc-drivers/renesas/eval_ad7793ebz_rx62n.jpg
   :align: center

Reference Project Overview
--------------------------

The demo program samples channel 1 at 16.7 Hz. The ADC is configured to work in
bipolar mode and the gain of the In-Amp is set to 1. The hex values of the
configuration and data registers are displayed on the LCD. The value of the Data
register is also formatted to be displayed in mV.

.. image:: https://wiki.analog.com/_media/resources/tools-software/uc-drivers/renesas/eval_ad7793ebz_rx62n_screen.jpg
   :align: center
