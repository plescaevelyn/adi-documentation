Custom Platform vs Standard Platforms
=====================================

\*\* Custom Platform \*\*

Custom platform in SigmaStudio+ gives the ability to independently create and assemble different re-usable processors as per the user requirements, defining the memory size and other processing abilities. Custom platforms can also be configured with .shape file using the 'LoadShape contents' option to re-use any configured shape into the system (refer the images below).

.. image:: https://wiki.analog.com/_media/resources/tools-software/sigmastudiov2/usingsigmastudio/custom.png
   :width: 600px

.. image:: https://wiki.analog.com/_media/resources/tools-software/sigmastudiov2/usingsigmastudio/customcontent.png
   :width: 600px

\*\* Standard Platform \*\*

Standard platforms in SigmaStudio+ are the platforms which are given by the application by default where in the contents of the platform are already assembled. However, users can configure the settings through the settings window. Users can check all the supported platforms :doc:`here </wiki-migration/resources/tools-software/sigmastudiov2/supportedplatforms>`.
