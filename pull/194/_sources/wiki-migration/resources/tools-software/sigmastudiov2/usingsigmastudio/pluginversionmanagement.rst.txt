Plugin Version Management
=========================

SigmaStudio+ supports manually upgrading the plugin versions through the plugin
version management.This is available under Tools as Project version control
.When the user selects the plugin version management under project version
control ,It will list all the plugins in the current project for which higher
versions are available.

The list may include details like the plugin name,current version,available
version,version description and update option of the higher version of the
plugins if available. When the user clicks on the upgrade option,the plugins
will be updated to the selected version.Only Upgrading of the plugins is
supported with the plugin version management,downgrading is not
supported.Individual plugins can be downgraded using the properties window.
