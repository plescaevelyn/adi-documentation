Accessing Help
==============

Users can access help through the ‘Help’ option in SigmaStudio+ application.
‘Index’ option under ‘Help’ menu launches the SigmaStudio+ Documentation landing
page. Select a given block on the canvas and go to ‘Help’ and click on ‘Content’
(refer the image below) to know more about the selected block. Alternatively,
users can also select the module and press “F1” to open the help page of that
selected block.
