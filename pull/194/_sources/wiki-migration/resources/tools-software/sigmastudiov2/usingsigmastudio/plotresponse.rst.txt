Plot Frequency and Phase response
=================================


This section provides the steps to generate frequency and phase response.

--------------

The users can plot frequency and phase response by using Stimulus and Probe modules in SigmaStudio+ application.

Connect the input of the algorithm you want to plot the response with Stimulus and out with probe. Click on generate button of Stimulus (which will generate the transfer function of the algorithm) and then the users can plot the frequency and phase response by clicking on "Show Graph" button in probe module (refer the image below).

.. image:: https://wiki.analog.com/_media/resources/tools-software/sigmastudiov2/usingsigmastudio/plot.png
   :width: 600px

Alternatively, users can also just do "shift" + hover over the algorithm to plot the frequency and phase response (refer the image below).

.. image:: https://wiki.analog.com/_media/resources/tools-software/sigmastudiov2/usingsigmastudio/plot1.png
   :width: 600px
