Update Manager
==============

Users can access the SigmaStudio+ updates from the SS+ **Update Manager**. Update Manager can be launched by clicking “\ **Check for Updates**\ ” from the '**Help**' menu. Update Manager will launch in a new window.

Update Manager displays all the updates available for the current version of SigmaStudio+ and its co-existing plugins such as A2B, if A2B products are installed. The description on the right side of the window will display the details of the selected update.

Click on the ‘\ **Install**’ button to install the selected update. When the Install button is clicked, a popup will appear asking the user permission to close SigmaStudio+ automatically. When ‘**Yes**’ is selected by the user, all open instances of SigmaStudio+ will be closed and the update operation will start. On the other hand, the update operation will not be performed if ‘**No**\ ’ is selected. The update Manager will relaunch SigmaStudio+ after completing the update process.
