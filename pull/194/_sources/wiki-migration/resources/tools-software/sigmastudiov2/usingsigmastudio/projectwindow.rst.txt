Project Window
==============

\*\* Users can perform multiple operations from the project window while
building the diagram: \*\*

- Navigate to the canvas, settings and algorithm with one click.(refer the image
  below).

.. image:: https://wiki.analog.com/_media/resources/tools-software/sigmastudiov2/usingsigmastudio/project.png
   :width: 400

- Perform link, compile, download by selecting the desired platform from the
  project window. Users can also rename the platform name by clicking on the
  rename button (refer the image below).

.. image:: https://wiki.analog.com/_media/resources/tools-software/sigmastudiov2/usingsigmastudio/lcdproject.png
   :width: 400

\*Navigate to the module in the schematic by double clicking on that module from the project window.

.. image:: https://wiki.analog.com/_media/resources/tools-software/sigmastudiov2/usingsigmastudio/navigateproject.png
   :width: 700
