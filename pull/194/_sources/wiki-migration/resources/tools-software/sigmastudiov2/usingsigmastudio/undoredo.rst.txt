Undo/Redo
=========

SigmaStudio+ tool supports the option of undo and redo of elements on the canvas in order to improve extensive workflows. In SigmaStudio+ it is possible to undo and redo contents present in the canvas. These actions can be achieved through different ways. In the first method, users can undo and redo contents through using key combination from the keyboard: ctrl + z for undo and ctrl + y for redo. In the second method, users can perform these actions through the undo and redo buttons provided in the application(refer the image below)

|image1| |image2|

Users can refer to the Undo History tab(refer the image below) for the list actions performed by them in a particular schematic which can be undone and redone.
