Schematic Version Control
=========================

Schematic version control allows the users to maintain version numbers for a particular schematic. This feature is disabled by default. This feature can be enabled from the top menu toolbar, Tools -> Version Control option. On clicking version control from the menu, the version control window opens up. The window has an option for enabling version control (which will be unchecked by default). The figure below depicts the version control window.

.. image:: https://wiki.analog.com/_media/resources/tools-software/sigmastudiov2/usingsigmastudio/versioncontrol.png
   :width: 600px

The version control window depicts the different versions associated with the particular schematic. The details include the version number, date, time and the description which was given by the user while saving the particular version. The version is incremented whenever there is a change in the schematic (code change) and the compilation is done.
