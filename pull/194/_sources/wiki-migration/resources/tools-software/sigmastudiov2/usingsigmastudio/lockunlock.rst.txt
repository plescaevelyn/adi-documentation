Lock/Unlock Shape
=================

On the SigmaStudio+ schematic view, a shape can be locked from the context menu with a password and then unlocked with the same password. The feature is useful in cases where in the user wants to protect their IP when sharing with multiple customers.

.. image:: https://wiki.analog.com/_media/resources/tools-software/sigmastudiov2/usingsigmastudio/lock.png
   :width: 400px

To unlock the shape use the 'Unlock Shape' in the context menu, note that no other options can be performed on the shape when it is locked except for the platform level operations.

.. image:: https://wiki.analog.com/_media/resources/tools-software/sigmastudiov2/usingsigmastudio/unlock.png
   :width: 400px
