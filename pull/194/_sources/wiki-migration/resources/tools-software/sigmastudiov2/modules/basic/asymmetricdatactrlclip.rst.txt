Asymmetric Soft Clipper - Data Controlled
=========================================

.. image:: https://wiki.analog.com/_media/resources/tools-software/sigmastudiov2/modules/basic/asymmclipperdatacontrolled.png
   :alt: asymmclipperdatacontrolled.png

Description
-----------

The Asymmetric Data Controlled Clipper is a soft-clipper that clips the input data signal according to threshold values set by data control input pins.
