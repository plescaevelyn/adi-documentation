Asymmetric Soft Clipper - Standard Cubic
========================================

.. image:: https://wiki.analog.com/_media/resources/tools-software/sigmastudiov2/modules/basic/asymmclippercubic.png
   :alt: asymmclippercubic.png

Description
-----------

The Asymmetric Clipper Cubic is a soft-clipper that clips the input data signal
according to clip up/down values set by data control input pins.
