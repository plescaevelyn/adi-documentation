Soft Clipper - Data Controlled
==============================

.. image:: https://wiki.analog.com/_media/resources/tools-software/sigmastudiov2/modules/basic/softclipdatacontrolled.png

Description
-----------

This block clips off portions of signal voltages above and/or below certain
limits.
