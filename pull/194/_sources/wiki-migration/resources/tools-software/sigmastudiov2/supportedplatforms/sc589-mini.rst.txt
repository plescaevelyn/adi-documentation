ADZS-SC589-MINI
===============

Evaluation Hardware for the ADSP-SC58x SHARC Family

.. container:: centeralign

   \ |image1|\


.. container:: centeralign

   \ **Figure:** ADSP-SC589-MINI


For Sport configuration, please refer example schematic of ADSP-SC589Mini in the :doc:`Tutorial and Examples </wiki-migration/resources/tools-software/sigmastudiov2/tutorials>` wiki page.

.. note::

   The ADSP-SC589 Mini board version 2.1 is supported from SigmaStudio+ version 2.3.0 by default. To use the older version of ADSP-SC589 Mini boards, the “\ ADSPSC589_SAM_BOARD_2_1\ ” macro in the ADSP-SC589-SAM demo application can be disabled using CCES project compiler properties.
