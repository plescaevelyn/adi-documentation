USBi (EVAL-ADUSB2EBZ)
=====================

The Analog Devices USBi “USB Interface” (EVAL-ADUSB2EBZ) board is the interface between PC's USB port and target platform's data input pins. The EVAL-ADUSB2EBZ board is capable of both SPI and I²C communication.

More details - :adi:`EVAL-ADUSB2EBZ`
