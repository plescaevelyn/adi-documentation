AARDVARK
========

.. image:: https://wiki.analog.com/_media/resources/tools-software/sigmastudiov2/supportedplatforms/aardvark.jpg
   :width: 400px

The Aardvark I2C/SPI Host Adapter is a fast and powerful I2C bus and SPI bus host adapter through USB. It allows the user to interface the PC running SigmaStudio+ via USB to the target platform and transfer serial messages using the I2C and SPI protocols.

More details - `Aardvark I2C/SPI Host Adapter <https://www.totalphase.com/products/aardvark-i2cspi/>`_
