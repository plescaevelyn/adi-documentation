Toolbox Window
==============

|image1| |image2|

The figures above depict the tree tool box window. There are two different toolbox windows displayed. The first window is displayed when a new project is created. It displays the list of platforms for which the project being created is targeted and the list of communication platforms available like usbi and aardvark. Using these blocks the main system configuration is setup.

The second tree tool box is displayed once the project is created and the desired platform is chosen and when the canvas for creating the schematic is opened. This Tree Toolbox window displays the available building blocks in a tree view. System blocks are arranged in a hierarchy of folders according to function and algorithm. Navigate the folders to find a desired algorithm block and drag and drop it onto the canvas. For more details on the available blocks, please refer to the modules section.
