Schematic Window
================

.. image:: https://wiki.analog.com/_media/resources/tools-software/sigmastudiov2/developmentenvironment/schematicwindow/schematicswindow.png
   :width: 400

The schematic window contains the link to all the schematics present in the
project. Instead of iterating through the project window to find the different
schematics, this window acts as a shortcut having links to all the schematics
present. The schematic window is docked horizontally right and vertically at the
center of the screen by default. The visibility of the window can be changed
based on users requirement from the top menu.

There are two icons depicted to the left of the schematic name. The first icon
opens the settings page of the schematic and the second icon opens the canvas on
which the schematic can be designed.
