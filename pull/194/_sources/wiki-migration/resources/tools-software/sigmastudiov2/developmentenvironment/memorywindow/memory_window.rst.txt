Memory Window
=============

.. image:: https://wiki.analog.com/_media/resources/tools-software/sigmastudiov2/developmentenvironment/memorywindow/memorywindow.png
   :width: 1000px

The memory window displays the memory related information for every module in the schematic. The dropdown on the top allows the user to select the schematic for which the memory details have to be displayed. Inside the window, for every module present in the chosen schematic details of module type, Uid and memory which include the sizestate32mem, sizestate48mem, sizeparammem, sizecircmem, sizepremem etc are displayed and the last row displays the total memory which is the summation of all the individual modules.

The memory window has two buttons on the top right corner. Their description is given below :

-  Copy - Copies the contents of the memory window to clipboard, which can be pasted as plain text in any editor.
-  Save - Save the contents of the memory window to a file
