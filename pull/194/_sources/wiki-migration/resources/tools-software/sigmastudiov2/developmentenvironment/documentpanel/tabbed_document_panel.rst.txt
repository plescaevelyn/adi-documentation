Tabbed Document Panel
=====================

.. image:: https://wiki.analog.com/_media/resources/tools-software/sigmastudiov2/developmentenvironment/documentpanel/tabbeddocpanel.png
   :width: 800

The figure above depicts the tabbed document panel. On top, the different tabs
which are open are displayed, and below it is the canvas on which the schematic
will be designed and implemented. Modules from the toolbox can be dragged and
dropped onto the empty canvas. Modules can be dragged in all directions and the
canvas scrolls automatically in the direction of drag. The bottom right of
canvas has a mini map button which depicts the compressed version of the entire
schematic built on the canvas. Using the minimap button the user can scroll to
the desired region on the canvas.
