Side Menu
=========

.. image:: https://wiki.analog.com/_media/resources/tools-software/sigmastudiov2/developmentenvironment/sidemenu/savemenu.png
   :width: 100px

The figure above depicts the side menu in SigmaStudio+. The side menu is a kind of shortcut menu which consists of buttons for enabling and disabling the visibility of the different panels present. Also there are buttons present to switch between tune and design mode. This switching is to ensure that the visibility of tabs and windows are taken care based on the mode of operation.

-  First Button corresponds to the design layout, which on clicking adjusts the view of SigmaStudio+ i.e. the windows and tabs which is suitable for the design mode which is necessary for building or editing schematics.
-  Second button is associated with the tune layout, which adjusts the view necessary when tuning the downloaded schematic.
-  The rest of buttons in the panel are used for enabling and disabling the visibility of different windows present in SigmaStudio+
