Properties Window
=================

.. image:: https://wiki.analog.com/_media/resources/tools-software/sigmastudiov2/developmentenvironment/propertieswindow/propertieswindow.png
   :width: 300

The properties window displays all the details related to any element that is
present on the canvas. The properties window is docked at the bottom right by
default when SigmaStudio+ is launched. When a particular element on the canvas
is selected, all the details of the module which include name, uid, version,
processing type, blocksize, sampling rate, number of channels or stages if any
and bypass options are displayed. The figure above depicts the details of
dynamic bass boost mono module instance.

The option of bypassing the module, excluding the export and changing the
precision type, number of channels, stages can be done through the properties
window. These options are also available in the context menu which appears upon
right click on the module.
