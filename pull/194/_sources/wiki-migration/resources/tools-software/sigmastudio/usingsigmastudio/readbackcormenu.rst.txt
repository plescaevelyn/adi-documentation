DSP Readback
------------

The DSP Readback block lets you read values back from the DSP at any point in your schematic design.

The number displayed onscreen is the data value sent back from the DSP considering all the blocks to the left of the Readback block. Every time you click Read, this value will be updated with the latest from the DSP. By displaying the output value from any block, in any format desired, Readback is used chiefly for debugging, and probably will prove very handy.

Enable All Continuous Readback Controls
---------------------------------------

Though, Readback Controls having switch to "ON/OFF" reading values from DSP in real time, user has to turn ON Readback Controls one by one by manually clicking "ON" button of each Readback Controls. But, this "Enable All Continuous Readback Controls" option will automatically "ON" All Readback Controls in the schematic by either clicking "Enable All Continuous Readback Controls" option in "Actions" menu bar or by using "ctrl+F11" keyboard shortcut.

.. image:: https://wiki.analog.com/_media/resources/tools-software/sigmastudio/usingsigmastudio/readbackenable.png
   :align: center

Disable All Continuous Readback Controls
----------------------------------------

Though, Readback Controls having switch to "ON/OFF" reading values from DSP in real time, user has to turn OFF Readback Controls one by one by manually clicking "OFF" button of each Readback Controls. But, this "Disable All Continuous Readback Controls" option will automatically "OFF" All Readback Controls in the schematic by either clicking "Disable All Continuous Readback Controls" option in "Actions" menu bar or by using "ctrl+F12" keyboard shortcut.

.. image:: https://wiki.analog.com/_media/resources/tools-software/sigmastudio/usingsigmastudio/readbackdisable.png
   :align: center
