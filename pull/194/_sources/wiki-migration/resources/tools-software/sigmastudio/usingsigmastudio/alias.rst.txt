Alias
=====

:doc:`Click here to learn basics about Alias </wiki-migration/resources/tools-software/sigmastudio/usingsigmastudio/buildingschematics/wiresandaliases>`

Add Alias
---------

Right clicking on Input Alias will show option to Add output Alias. That output
Alias can be moved to anywhere in schematic and connected with any block as
signal destination. One Input Alias can have multiple output Aliases if
required.

.. image:: https://wiki.analog.com/_media/resources/tools-software/sigmastudio/usingsigmastudio/alias_addalias.png
   :align: center

Highlighting All Alias
----------------------

Clicking on any Input/Output Alias will highlight All Aliases. Output Aliases
will be highlighted in "Aqua" color and Input Alias will be highlighted in
"Orange" color.

.. image:: https://wiki.analog.com/_media/resources/tools-software/sigmastudio/usingsigmastudio/alias_highlight.png
   :align: center

Go to CounterPart Alias
-----------------------

User can go to particular output alias of choice by using this option. Example:
If user clicks "1" in Goto CouterPart Alias option, then "FIR1" block will come
into center part of schematic even if it is not in visible area of schematic.
This helps user to easily navigate between and evaluate the schematic design.

.. image:: https://wiki.analog.com/_media/resources/tools-software/sigmastudio/usingsigmastudio/alias_gotooutputalias.png
   :align: center

Go to Input Alias
-----------------

At any time, user can navigate back to Input Alias from any Output Alias using
this option. Goto Input Alias option will be shown in every output alias. After
clicking that option, Input Alias block will come into center part of schematic
even if it is not in visible area of schematic. This helps user to easily
navigate between and evaluate the schematic design.

.. image:: https://wiki.analog.com/_media/resources/tools-software/sigmastudio/usingsigmastudio/alias_gotoinputalias.png
   :align: center
