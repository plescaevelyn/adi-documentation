About Dialog
============

:doc:`Click here to return to the Dialogs page </wiki-migration/resources/tools-software/sigmastudio/developmentenvironment/dialogs>`

The About dialog is accessed by choosing **H\ elp - A\ bout** from the main application menu. It displays the program version and copyright information. If you are reporting SigmaStudio issues to Analog Devices support, please include this version information with any written communication.
