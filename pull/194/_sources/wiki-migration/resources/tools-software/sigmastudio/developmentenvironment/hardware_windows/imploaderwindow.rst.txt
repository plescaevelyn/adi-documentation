Imploader
=========

:doc:`Click here to return to the Hardware Windows page </wiki-migration/resources/tools-software/sigmastudio/developmentenvironment/hardware_windows>`

The Imploader window (**AD1940 and AD1941 only**) allows you to compile and download previously compiled projects without the need to load the project files. This is useful for comparing among different settings or flows.

To access the Imploader window:

-  Click the Hardware Configuration tab at the top of the workspace.
-  Right-click on the DSP processor block to display the context menu.
-  Select **Imploader Window** from the menu.

To load a program or parameters:

-  Select **File - Open** from the menu or click the Open toolbar button.
-  Locate the compiled netlist file for the project you want to load (\*.cir, \*.cir2) and click open.
-  Press the "Download Program and Parameters" button.
