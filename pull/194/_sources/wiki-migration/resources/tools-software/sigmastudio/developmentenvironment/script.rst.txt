Script Dialog
=============

:doc:`Click here to return to the Dialogs page </wiki-migration/resources/tools-software/sigmastudio/developmentenvironment/dialogs>`

SigmaStudio includes a script-able automation engine allowing control over most common application operations. To access the Script dialog, select **Tools - Script...** in the main application menu or press F12.

Scripting in both the C# and Visual Basic languages is supported. For more information about the SigmaStudio Script engine check the detailed documentation in the attached PDF: `sigmastudio_scripting.pdf <https://wiki.analog.com/_media/resources/tools-software/sigmastudio/developmentenvironment/sigmastudio_scripting.pdf>`_

SigmaStudioServer:
------------------

SigmaStudioServer is an automation interface for SigmaStudio that allows external software applications to control or automate SigmaStudio functions. (For example, National Instruments LabVIEW) The server interface library Analog.SigmaStudioServer.dll implements Microsoft .Net and ActiveX server interfaces. For more information or detailed documentation please contact Analog Devices on the :ez:`SigmaDSP Community on EngineerZone <community/dsp/sigmadsp>`.
