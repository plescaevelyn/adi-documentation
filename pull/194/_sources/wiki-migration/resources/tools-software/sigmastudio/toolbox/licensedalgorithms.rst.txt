Licensed Algorithms
===================

:doc:`Click here to return to the Toolbox page </wiki-migration/resources/tools-software/sigmastudio/toolbox>`

The Toolbox's Licensed Algorithms library lets you access proprietary licensed blocks (including some developed by Analog Devices) for use in your designs.

The following Algorithms are available:

-  **Analog Devices**

   -  :doc:`ADI Surround </wiki-migration/resources/tools-software/sigmastudio/toolbox/licensedalgorithms/adisurround>`
   -  :doc:`ADI Virtual </wiki-migration/resources/tools-software/sigmastudio/toolbox/licensedalgorithms/adivirtual>`
   -  :doc:`External LED Array Driver </wiki-migration/resources/tools-software/sigmastudio/toolbox/licensedalgorithms/externalledarraydriver>`
   -  :doc:`Subharmonic Synth (general) </wiki-migration/resources/tools-software/sigmastudio/toolbox/licensedalgorithms/subharmonicsynthgeneral>`
   -  :doc:`Subharmonic Synth (LF) </wiki-migration/resources/tools-software/sigmastudio/toolbox/licensedalgorithms/subharmonicsynthlf>`

-  **Kronoton GmbH**

   -  :doc:`HDSX® (High Definition Sound Expansion) </wiki-migration/resources/tools-software/sigmastudio/toolbox/licensedalgorithms/hdsx>`

Additional algorithms may be available by request from 3rd Party vendors.

.. note::

   Note: None of these algorithm blocks are included with the SigmaStudio application installation. To evaluate or license algorithms from third parties, please contact their licensing departments directly.
