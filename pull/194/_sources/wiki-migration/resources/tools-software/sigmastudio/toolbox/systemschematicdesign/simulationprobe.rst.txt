Simulation Probe
================

:doc:`Click here to return to the System page </wiki-migration/resources/tools-software/sigmastudio/toolbox/systemschematicdesign>`

|simprobepic1.png| The Simulation Probe is used in conjunction with the :doc:`Simulation Stimuli </wiki-migration/resources/tools-software/sigmastudio/toolbox/systemschematicdesign/simulationstimuli>` to plot the frequency response of the system you configure. Click the Probe **block** to bring up the graph (blank), then the Stimulus block to show the curve for your system. See the :doc:`System Example </wiki-migration/resources/tools-software/sigmastudio/tutorials/systemexamples>` and :doc:`second tutorial </wiki-migration/resources/tools-software/sigmastudio/tutorials/probeandstimulusblocks>` for more information.
