Level Detectors / Lookup Tables
===============================

:doc:`Click here to return to the Toolbox page </wiki-migration/resources/tools-software/sigmastudio/toolbox>`

The Level Detectors / Lookup Tables library of the ToolBox gives you access to
blocks which provide graphical representations of signal level.

The following are available:

-  :doc:`2D Index Lookup Table </wiki-migration/resources/tools-software/sigmastudio/toolbox/leveldetectorslookuptables/2dindexlookuptable>`
-  :doc:`Index Lookup Table </wiki-migration/resources/tools-software/sigmastudio/toolbox/leveldetectorslookuptables/indexlookuptable>`
-  :doc:`Level Detector Designer </wiki-migration/resources/tools-software/sigmastudio/toolbox/leveldetectorslookuptables/leveldetectordesigner>`
-  :doc:`RMS Table </wiki-migration/resources/tools-software/sigmastudio/toolbox/leveldetectorslookuptables/rmstable>`
-  :doc:`Single Band Level Detectors </wiki-migration/resources/tools-software/sigmastudio/toolbox/leveldetectorslookuptables/singlebandleveldetector>` (includes "Numerical Display", "Running Average", "Direct Read", and "Low DSP MIPS" blocks)
-  :doc:`Seven Band Level Detector </wiki-migration/resources/tools-software/sigmastudio/toolbox/leveldetectorslookuptables/sevenbandleveldetector>`
-  :doc:`Signal Detection </wiki-migration/resources/tools-software/sigmastudio/toolbox/leveldetectorslookuptables/signaldetection>`

The topic pages contain detailed information, but also look at the :doc:`Level Detectors Examples </wiki-migration/resources/tools-software/sigmastudio/tutorials/leveldetectorsexamples>` for sample schematics.
