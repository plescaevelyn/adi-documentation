Introduction
============

This document provides a comprehensive guide to building and running the A2B
Stack Software and application on a Raspberry Pi, specifically using the
Raspberry Pi 4 with the I2S protocol. The application is created by
cross-compiling the A2B stack for the AARCH64 platform, allowing it to run
efficiently on the Raspberry Pi.

Scope
-----

The scope of this document is to provide steps to build and run the A2B Stack
Software and application on a Raspberry Pi. Additionally, the document covers
the various features of the application and the A2B Stack Software on Raspberry
Pi.
