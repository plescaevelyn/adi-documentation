Raspberry Pi4 Model B
=====================

Download the Raspberry Pi Imager application from the following link `Raspberry Pi Imager <https://www.raspberrypi.com/software/>`_ and install it on your computer. Open the application and select your Raspberry Pi board. Choose “Raspberry Pi OS (64-bit)” as the operating system and pick the storage device. Finally, write the OS to the storage device to prepare it for use as shown in the image below.
