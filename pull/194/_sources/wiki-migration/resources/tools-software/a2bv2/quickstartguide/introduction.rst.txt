Introduction
============

Purpose
-------

This document enables the user to evaluate A2B network from SigmaStudio+ tool.

Scope
-----

This guide details the steps for setting up and running the sample demo. Prior
knowledge of using the SigmaStudio+ tool is recommended.

Organization of the document
----------------------------

Section :doc:`1 </wiki-migration/resources/tools-software/a2bv2/quickstartguide/introduction>` is the introduction.

Sections :doc:`2 </wiki-migration/resources/tools-software/a2bv2/quickstartguide/systemrequirements>` and :doc:`3 </wiki-migration/resources/tools-software/a2bv2/quickstartguide/hardware-setup>` contains the various requirements and instructions to set up the sample demo system.

Section :doc:`4 </wiki-migration/resources/tools-software/a2bv2/quickstartguide/running-sample-demo>` explains the steps for running the demo.

Appendices :doc:`Capture window and sequence window </wiki-migration/resources/tools-software/a2bv2/quickstartguide/appendix-f>` gives a brief overview of using A2B in SigmaStudio+.

Appendix :doc:`ADSP-2156x </wiki-migration/resources/tools-software/a2bv2/quickstartguide/adsp-21569_ez-kit>` has instructions to build an ADSP-2156x target project.

Appendix :doc:`Schematics of codecs </wiki-migration/resources/tools-software/a2bv2/quickstartguide/appendix-k>` talks about the Schematics of Codecs used in EVAL-AD2437 Platforms.

Appendix :doc:`Two step discovery </wiki-migration/resources/tools-software/a2bv2/quickstartguide/appendix-l>` explains Two Step Discovery for AD2437 Platforms.
