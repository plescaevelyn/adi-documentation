APPENDIX E: Plugin Upgrades
===========================

Plugins can be upgraded to a higher version (if it exists) using either of the
following methods

Plugin upgrades
---------------

Project version control
~~~~~~~~~~~~~~~~~~~~~~~

-  On the toolbar click on Tools -> project version control. A version control tab will open up.
-  Select plugin version management.
-  A list of plugins will be listed in this window as shown in :doc:`Figure </wiki-migration/resources/tools-software/a2bv2/quickstartguide/appendix-i>`.
-  Select an available plugin version and click on “upgrade”.

.. image:: https://wiki.analog.com/_media/resources/tools-software/a2bv2/quickstartguide/plugin_version_upgrade_through_version_control.png
   :align: center

.. container:: centeralign

   \ **Figure:** Plugin version upgrade through version control

.. note::

   Upgrade of AnalogDevices.Plugins.Channels.A2BChannel is not supported.

Properties window
~~~~~~~~~~~~~~~~~

-  Single left-click any plugin on the canvas
-  As shown in :doc:`Figure </wiki-migration/resources/tools-software/a2bv2/quickstartguide/appendix-i>`, on the properties window towards the right, select the required version from the drop-down next to Version.

   :align: center

.. container:: centeralign

   \ **Figure:** Plugin version upgrade through Properties window
