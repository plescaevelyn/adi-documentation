Capture Window & Sequence Window
================================

Capture Window
--------------

:doc:`Capture window </wiki-migration/resources/tools-software/sigmastudiov2/developmentenvironment/capturewinow/capture_window>` will captures all the commands that exchanges between PC to the A2B network. For example, please refer the below snapshot.

|image1|

.. container:: centeralign

   **Figure:** Capture Window

It also has features as mentioned below:

-  copying all the commands
-  copying the selected commands
-  copying the commands to “Sequence” window

Sequence Window
---------------

:doc:`Sequence window </wiki-migration/resources/tools-software/sigmastudiov2/developmentenvironment/sequencewindow/sequence_window>` has features as mentioned below:

-  to import the xml’s file (ex: command list xml file) and can be downloaded.
-  save the sequence as a xml file
-  multiple sequences can be created

   **Figure:** Sequence Window
