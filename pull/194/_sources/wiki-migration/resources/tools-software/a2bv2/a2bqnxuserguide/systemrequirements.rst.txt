System Requirements
===================

This section mentions the hardware and software requirements for running the
demo on BeagleBone Black platform.

Hardware Requirements
---------------------

PC
~~

-  Windows Host

A2B Evaluation Hardware
~~~~~~~~~~~~~~~~~~~~~~~

-  BeagleBone Black PCB Rev B6

   -  Evaluation Board

      -  1 Ethernet cable – RJ45 Cat5E
      -  USB Micro-B to Standard-A cable
      -  USB to UART converter
      -  Micro -SD card (4GB or higher)

-  EVAL-AD2428WD1BZ Rev1.1 package

   -  Evaluation Board

      -  12V Power Supply 1.5A
      -  USB Type A Male - Mini Type B Male cable
      -  Twisted Pair A2B Cable with DuraClick connectors
      -  ADUSB2EBZ (USBi) board with USB cable (1.7m or 10m)
      -  3.5mm Audio cable or male-male adapte

-  EVAL-AD2428WB1BZ Rev2.1 package

   -  Evaluation Board EVAL-AD2428 WB1BZ (Stereo In, Stereo Out, Stereo
      Microphones)

      -  Twisted Pair A2B Cable with DuraClick connectors (30cm or 4m)
      -  3.5mm Audio cable or male-male adapter

-  EVAL-AD2428WC1BZ Rev2.1 package

   -  Evaluation Board EVAL-AD2428 WC1BZ (4 Microphones)

      -  Twisted Pair A2B Cable with DuraClick connectors (30cm or 4m)

-  One audio sink devices (e.g. Speakers)
-  Fly Wires 3-nos
-  USB to TTL Serial cable

Software Requirements
---------------------

Windows Host
~~~~~~~~~~~~

-  QNX Software Center v1.3.1
-  QNX Software Development Platform 7.0
-  QNX Momentics v 7.1.2
-  QNX SDP 7.0 Board Support Package for BeagleBone Black or custom platform.
-  TeraTerm/Putty or any other terminal application for serial connection to the target
-  Java Runtime Environment for Sequence chart feature as explained in :doc:`Sequence Chart </wiki-migration/resources/tools-software/a2bv2/a2bqnxuserguide/demoapplicationandtrace>`

.. note::

   QNX Software Center and QNX Momentics are installed from a single installer.
