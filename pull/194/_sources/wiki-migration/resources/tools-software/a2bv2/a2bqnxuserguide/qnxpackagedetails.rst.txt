A2B Stack on QNX Package Details
================================

The release package contains folder structure as shown in below Figure.

.. image:: https://wiki.analog.com/_media/resources/tools-software/a2bv2/a2bqnxuserguide/qnx_package_details.png
   :align: center
   :width: 400px

The files specific to the demo application on QNX are located in the ADI_A2B_Software-QNX- RelX.Y.Z/Target/examples/demo/a2b-qnx folder. The various folders in the a2b-qnx folder are explained in below table.

+---------------------------------------+---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------+
| **Folder Name**                       | **Purpose**                                                                                                                                                                                                                                                                                                                           |
+=======================================+=======================================================================================================================================================================================================================================================================================================================================+
| a2bapp-qnx                            | This folder contains the QNX Momentics project for A2B Stack Software demo application in QNX running on BeagleBone Black board.                                                                                                                                                                                                      |
+---------------------------------------+---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------+
| a2bapp-qnx/app                        | This folder contains the files for the application layer and SigmaStudio exported bus config file.                                                                                                                                                                                                                                    |
+---------------------------------------+---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------+
| a2bapp-qnx/a2bstack-pal               | This folder contains the PAL files for A2B stack on QNX.                                                                                                                                                                                                                                                                              |
+---------------------------------------+---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------+
| a2bapp-qnx/a2bstack- pal/platform/a2b | This folder contains header files for the platform abstraction layer on QNX. These files may need to be modified according to the platform specifics. Refer to :doc:`Build for a different platform </wiki-migration/resources/tools-software/a2bv2/a2bqnxuserguide/a2bstackdemoapplicationbuild>` for more information.              |
+---------------------------------------+---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------+
| Makefiles_Windows                     | This folder contains the Makefiles for building the A2B application                                                                                                                                                                                                                                                                   |
+---------------------------------------+---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------+
