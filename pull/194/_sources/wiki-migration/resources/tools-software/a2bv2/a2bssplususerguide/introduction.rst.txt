Introduction
============

This document provides an overview on how to draw A2B schematics on
SigmaStudio+. The document describes about various features available in
SigmaStudio+ for A2B network discovery, configuration, debug, and analysis.
Using SigmaStudio+ one can quickly prototype and evaluate an A2B system on a
Host PC. Once the system is verified, the network configuration information can
be exported in various formats for easy integration into an embedded platform.
