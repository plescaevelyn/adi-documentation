ADALM1000 and PixelPulse Software Overview

The ADALM1000 (M1K) and PixelPulse application software provide an experimenter with a portable lab environment that can source and measure voltage and current signals on two channels.

Operational Modes

Each channel can operate in three distinct modes:

  1. Measure Voltage - Functions similarly to an oscilloscope, displaying waveforms on the computer screen.

  2. Source Voltage / Measure Current - Sources one of several voltage waveforms while measuring the current through the voltage source.

  3. Source Current / Measure Voltage - Sources current while measuring voltage across the current source.

Flexible Configuration

The dual-channel design allows flexible configurations. For example, one channel can source voltage while another measures it, enabling V/I characteristic testing of components and circuits.

This makes the ADALM1000 ideal for:

  • Exploring passive component behavior (resistors, capacitors, inductors)

  • Testing semiconductor characteristics (diodes, transistors)

  • Analyzing circuit frequency response

  • Understanding power relationships
