DE10-Nano Quickstart

Requirements

Hardware

  1. EVAL-CN0540-ARDZ

  2. DE10-Nano on Arduino shield connector

    1. 5V/2A Wall Power supply with barrel jack (comes with DE10-Nano)

    2. mini USB to USB Type A (comes with DE10-Nano)

  3. Class 10 16GB SD Card

  4. Ethernet cable

  5. IEPE Compatible Sensor

Software

  • ADI Kuiper Image for CN0540

  • IIO-Oscilloscope

Setup

SD-Card

To prepare the SD-card for the DE10-Nano board:

  1. Download ADI Kuiper Image for CN0540

  2. Validate, Format, and Flash the SD Card

    1. Format and flash the SD Card using Windows

    2. Format and flash the SD Card using Linux

Once microSD card has been imaged, safely remove the hardware from the SD card writer, and insert the card directly into the microSD card slot on the DE10-Nano.
