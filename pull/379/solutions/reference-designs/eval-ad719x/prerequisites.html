Prerequisites

What you need depends on what you are trying to do. As a minimum, you need to start out with:

Hardware prerequisites

  1. One of the EVAL-AD719xASDZ evaluation boards:

    • EVAL-AD7190ASDZ

    • EVAL-AD7192ASDZ

    • EVAL-AD7193ASDZ

    • EVAL-AD7194ASDZ

    • EVAL-AD7195ASDZ

  2. A controller board. The supported options are:

    • EVAL-SDP-CK1Z (connects via 120-pin or Arduino connectors) with a USB-C cable

    • EVAL-SDP-CB1Z (connects via 120-pin connector only) with a Micro-USB cable

    • Cora Z7S (connects via Pmod JA or Arduino shield) with a Micro-USB cable

    • DE10-Nano (connects via Arduino shield) with a Micro-USB cable

  3. A PC running Windows with a USB 2.0 port.

  4. Micro SD card with at least 16 GB (required for FPGA carrier boards)

Warning

The evaluation software and drivers must be installed before connecting the evaluation board and controller board to the PC’s USB port, to ensure the system is correctly recognized.

Software prerequisites

If using ACE Software (SDP controller boards)

  1. ACE Software

  2. The ACE plugin matching your device:

    • AD7190 ACE Plugin

    • AD7192 ACE Plugin

    • AD7193 ACE Plugin

    • AD7194 ACE Plugin

    • AD7195 ACE Plugin

  3. SDP controller board drivers (required if not already installed on the PC)

If using IIO Oscilloscope (FPGA carrier boards)

  1. SD card imaged with Kuiper Linux

  2. IIO Oscilloscope

  3. A serial terminal (PuTTY, Tera Term, Minicom, etc.) configured at 115200 baud (8N1)
