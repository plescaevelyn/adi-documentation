Linux Device Drivers
====================

EVAL-CN0363-PMDZ uses Linux drivers in the
:dokuwiki:`IIO framework </software/linux/docs/iio/iio>`; it is possible to use
the
:dokuwiki:`IIO-Scope </resources/tools-software/linux-software/iio_oscilloscope>`
to access the raw sample data from the EVAL-CN0363-PMDZ. This is mainly intended
to be used for debugging purposes.

Here is the link to
:dokuwiki:`Linux device drivers </resources/tools-software/linux-drivers-all>`
for some Analog Devices products.

A detailed step on Zedboard driver installation for Debian is provided
`here <https://github.com/PyHDI/zynq-linux>`__.
