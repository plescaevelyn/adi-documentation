Precision Integrating TIA KWIK Lecture + Lab
============================================

General Description
-------------------

Learn to measure femtoamps in just a few minutes using an integrating
transimpedance amplifier.

The Precision Integrating Transimpedance Amplifier (TIA) Know-how with
Integrated Knowledge (KWIK) Demo provides a hands-on activity allowing the user
to measure femtoamp levels of dark current of a transimpedance amplifier with a
large area photodiode as well as pA levels of signal current by use of an
included LED and the ADALM2000 Active Learning Module.

The Precision Integrating TIA KWIK Demo uses an easy-to-use demonstration board,
specifically, EVAL-KW4101Z, that directly plugs into the ADALM2000.

Objective
---------

The Precision Integrating TIA KWIK Demo and Lecture and Lab aims to demonstrate
how to easily measure tiny currents, femtoamps, picoamps, etc. commonly
encountered with photodiode transimpedance amplifiers. The lecture and lab focus
exclusively on the operating principles of an integrating transimpedance
amplifier. The demonstration hardware (EVAL-KW4101Z) includes an LED light
source, a large area photodiode and integrating transimpedance amplifier. This
evaluation hardware is capable of demonstrating less than 0.5 fArms of input
referred dark current noise.
