EVAL-CN0363-PMDZ Linux software applications
============================================

The primary program to interact with the EVAL-CN0363-PMDZ is the
:dokuwiki:`ADI Colorimeter application </resources/tools-software/linux-software/colorimeter>`.
It can be used to gather, manage and compare processed sample data from the
EVAL-CN0363-PMDZ.

Since the Linux drivers used for the EVAL-CN0363-PMDZ use the
:dokuwiki:`IIO framework </software/linux/docs/iio/iio>` it is possible to use
the
:dokuwiki:`IIO-Scope </resources/tools-software/linux-software/iio_oscilloscope>`
to access the raw sample data from the EVAL-CN0363-PMDZ. This is mainly intended
to be used for debugging purposes.

All applications run directly on the ZED board itself.
