Wi-Fi Software Pack
===================

.. warning::

   **This CMSIS pack and its contents are DEPRECATED and no longer supported.
   Please do not use this moving forward**

General Description/Overview
----------------------------

Wi-Fi software pack contains source code written to interface the ADuCM3029 with
the ESP8266 Wi-Fi radio module.(Found on the EVAL-ADICUP3029) This software
package acts as a library that can be used with any platform containing the
ADuCM3029 and the ESP8266.

To use the ADI_WifiSoftware Pack with the ADICUP3029, you will also require the
following software, tools, CMSIS Packs:

- CrossCore Embedded Studio
- ADuCM302x Device Family Pack
- ADICUP3029 Board Support Package

For detailed information regarding the WiFi software pack, please see our complete WiFi software user guide. .. note::

   :git-EVAL-ADICUP3029:`Wi-Fi Software Pack 1.0.1 Release Notes <cmsis-packs/ADI-WifiSoftware/Documents+>`

.. important::

   You **MUST** have this software package installed on your laptop or PC in
   order to compile, debug, and run the applications using the WiFi for the
   ADICUP3029 platform.

Downloading the WiFi Software Pack
----------------------------------

The software pack can be downloaded in several ways.

#. Downloaded via the tools program

- It is recommended to download the WiFi software pack through from the tools
  program you are using. That way, all the files, directories structure, and
  project structure for the various applications is properly saved and accessed.
  For a detailed description on how to download the WiFi software pack through
  CrossCore Embedded Studio please see our
  :dokuwiki:`CrossCore Embedded Studio Quickstart User Guide </resources/eval/user-guides/eval-adicup3029/tools/cces_user_guide#how_to_install_or_upgrade_the_aducm302x_device_family_pack_dfp_for_cces>`.

#. Downloaded to local directory

- However if you do decide to download the WiFi software pack to your PC/laptop
  directly, please use the link below, and make sure you save the software pack
  to the correct local directory for your applications/projects.

   Download the WiFi Software Pack to your computer.
