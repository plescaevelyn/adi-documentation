Tools and Driver Details
========================

This chapter provides all the necessary steps to download, install, and setup
the software environment from Analog Devices. There is also information on the
different USB modes and drivers needed.

It contains four main sections:

-
  :dokuwiki:`CrossCore Download & Installation Guide </resources/eval/user-guides/eval-adicup360/tools/cces_setup_guide>`
  - Provides all the necessary instructions on how to download and install the
  CrossCore Embedded Studios software development environment.
-
  :dokuwiki:`CrossCore Quickstart User Guide </resources/eval/user-guides/eval-adicup360/tools/cces_user_guide>`
  - Provides information about using the CrossCore Embedded Studios IDE, in
  particular, the process of importing, building, debugging, and creating user
  applications for the EVAL-ADICUP360.
-
  :dokuwiki:`Hardware Drivers and USB Storage Drives </resources/eval/user-guides/eval-adicup360/tools/adicup360_hw_drivers>`
  - Provides detailed information how to load pre-compiled .HEX or .BIN files
  using the USB drive of the EVAL-ADICUP360 board.
-
  :dokuwiki:`Using EVAL-ADICUP360 with IAR and Keil IDEs </resources/eval/user-guides/eval-adicup360/tools/keil_iar_support>`
  - Provides detailed information how to use the EVAL-ADICUP360 board with IAR
  Embedded Workbench and Keil µVision.
