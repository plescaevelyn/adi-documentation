Hardware Cases and Enclosures
=============================

3D Printer Cases
----------------

There are enclosures made for the EVAL-ADICUP360 and attachable daughter boards.
I have put together a table of the current 3D printable cases we have to date.
If you have created a 3D case for the ADICUP360 and would like to have it added
to this table, please do so. We will review your submission and work on adding
it to the list.

.. tip::

   If you would like to view the designs in a 3D environment to get a better
   idea of what the design looks like, there are many free STL viewers that
   exist on the internet. I"ve provided a link below to the one that I think is
   the easiest to use, you can simply drag and drop the STL files into the
   browser.

   http://www.viewstl.com/

.. important::

   These 3D printer STL files were made by community contributors as additional
   collateral to the ADICUP360 ecosystem. Analog Devices is **NOT** responsible
   for 3D files that print incorrectly or are not compatible with your
   particular printer. It is up to the user to make sure the enclosures can be
   printed on their machines.
