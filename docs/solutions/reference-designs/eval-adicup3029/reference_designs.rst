Application Example Reference Designs
=====================================

This chapter contains various reference designs available for the base board and
the various add on modules. Each sub-section describes the demo program, how to
setup the hardware, how to obtain the source code, and finally, how to import
the project in the Eclipse workspace and to run it.

The following projects are currently available:

-  :doc:`Blinking LEDs Demo </solutions/reference-designs/eval-adicup3029/reference_designs/demo_blink>` - Shows the basic steps of creating a new project for the :doc:`EVAL-ADICUP3029 base board </solutions/reference-designs/eval-adicup3029/hardware/adicup3029>`, running and debugging the software.
-  :doc:`Command Line Interpreter Demo </solutions/reference-designs/eval-adicup3029/reference_designs/demo_cli>` - A Command Line Interpreter (CLI) demo project for the :doc:`EVAL-ADICUP3029 base board </solutions/reference-designs/eval-adicup3029/hardware/adicup3029>`.
-  :doc:`Accelerometer Demo using Wi-Fi </solutions/reference-designs/eval-adicup3029/reference_designs/demo_adxl362>` - Illustrates the functionality of the ADXL362 3-axes accelerometer. It works with the `EVAL-ADXL362-ARDZ Shield <https://wiki.analog.com/resources/eval/user-guides/eval-adicup360/hardware/adxl362>`_.
-  :doc:`Impact Measurement Demo using Bluetooth </solutions/reference-designs/eval-adicup3029/reference_designs/demo_adxl372>` - Illustrates the functionality of the ADXL372 3-axis high-g accelerometer. It works with the :doc:`EVAL-ADXL372-ARDZ Shield </solutions/reference-designs/eval-adicup3029/hardware/adxl372>`.
-  :doc:`PMOD Accelerometer Demo </solutions/reference-designs/eval-adicup3029/reference_designs/demo_pmodacl2>` - Illustrates the functionality of the ADXL362 3-axes accelerometer. It works with the `PmodACL2 from Digilent <http://store.digilentinc.com/pmod-acl2-3-axis-mems-accelerometer/>`_.
-  :doc:`Temperature Demo </solutions/reference-designs/eval-adicup3029/reference_designs/demo_adt7420>` - Weigh Scale measurement example for 4-/6-wire bridge sensors. It works with the `EVAL-ADT7420-PMDZ Shield <https://wiki.analog.com/resources/eval/user-guides/eval-adicup360/hardware/adt7420>`_.
-  :doc:`pH Measurement Demo </solutions/reference-designs/eval-adicup3029/reference_designs/demo_cn0326>` - pH Measurement System with Temperature Compensation that works with the `EVAL-CN0326-PMDZ PMOD <https://wiki.analog.com/resources/eval/user-guides/eval-adicup360/hardware/cn0326>`_.
-  :doc:`Toxic Gas(CO) Detection Demo using Bluetooth </solutions/reference-designs/eval-adicup3029/reference_designs/demo_cn0357>` - Measuring toxic gases using electrochemical sensors that work with the `EVAL-CN0357-ARDZ Shield <https://wiki.analog.com/resources/eval/user-guides/eval-adicup360/hardware/cn0357>`_ and sending it to a smart device via Bluetooth.
-  :doc:`Visible Light(RBG)Detection Demo </solutions/reference-designs/eval-adicup3029/reference_designs/demo_cn0397>` - Measuring visible light intensity and spectrum concentration levels without optical filters or other mechanical components using the `EVAL-CN0397-ARDZ Shield <https://wiki.analog.com/resources/eval/user-guides/eval-adicup360/hardware/cn0397>`_.
-  :doc:`Soil Moisture and pH Measurement Demo using Wi-Fi </solutions/reference-designs/eval-adicup3029/reference_designs/demo_cn0398>` - Measures the moisture level and and pH value of different kinds of soil using the `EVAL-CN0398-ARDZ Shield <https://wiki.analog.com/resources/eval/user-guides/eval-adicup360/hardware/cn0398>`_ and sending the data over Wi-Fi.
-  `BLE device Documentation <https://wiki.analog.com/resources/eval/user-guides/eval-adicup3029/reference_designs/ble_packet>`_ - Documentation referring to ADI BLE message format for use with IOTnode mobile application
-  :doc:`Optical Platform Demo </solutions/reference-designs/eval-adicup3029/reference_designs/demo_cn0503>` - Can measure different qualities of a liquid, such as colorimetry, fluorescence, turbidity, etc., using the `EVAL-CN0503-ARDZ Shield <https://wiki.analog.com/resources/eval/user-guides/circuits-from-the-lab/cn0503>`_.
