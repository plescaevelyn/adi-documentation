EVAL-CN0354-PMDZ
=================

Low Power 4-Channel Thermocouple Measurement.

Overview
--------

:adi:`CN0354` is a flexible, 4-channel, low power thermocouple measurement
circuit with an overall power consumption of less than 8 mW. The circuit has a
multiplexed front end, followed by an instrumentation amplifier that performs
cold junction compensation (0 deg C to 50 deg C) and converts the thermocouple
output to a voltage with a precise scale factor of 5 mV/°C. The error is less
than 2 deg C over a measurement range of -25 deg C to +400 deg C, and is
primarily due to the thermocouple nonlinearity. A nonlinearity correction
algorithm reduces the error to less than 0.5 deg C over a 900 deg C measurement
range. Noise-free resolution is better than 0.1 deg C.

The signal is then digitized by a 24-bit sigma-delta ADC, and the digital value
is provided on an I2C serial interface. With the PMOD form factor for rapid
prototyping, the design requires minimal PCB area and is ideal for applications
that require precise thermocouple temperature measurements.

The four thermocouple inputs to the circuit are terminated at the isothermal
block, **P2**. The :adi:`ADG1609` CMOS multiplexer switches the four
thermocouple channels to a single signal conditioning block to handle four
thermocouple inputs.

The signal is amplified by the :adi:`AD8495`, a precision instrumentation
amplifier that is laser trimmed to provide a precise 5 mV/°C output for a K
type thermocouple. The :adi:`AD8495` also provides cold junction compensation
over a range of 0 deg C to 50 deg C.

The fifth thermocouple is added to cancel the voltage generated by any
temperature differential that exists between the isothermal block and the
:adi:`AD8495` cold junction compensation circuit.

The :adi:`AD7787` is a 24-bit, low noise, low power sigma-delta ADC for low
frequency measurement applications, such as thermocouple measurement systems.
Its internal clock eliminates the need for an external clock and makes the
output data rate user configurable. This can reduce current consumption because
it functions at a lower internal clock frequency. It contains a sigma-delta ADC
with one differential input and one single-ended input, either of which can be
buffered or unbuffered after passing through a multiplexer.

Required Equipment
------------------

- :adi:`EVAL-SDP-CB1Z` controller board (SDP-B board)
- :adi:`SDP-I-PMOD <eval_sdp-pmod>` interposer board (SDP-PMOD interposer
  board)
- :adi:`EVAL-CN0354-PMDZ <CN0354>` evaluation board (CN-0354 board)
- :adi:`EVAL-CFTL-6V-PWRZ` +6 V power supply or equivalent DC power supply
- CN0354 Evaluation Software
- USB Type-A to USB Mini-B cable
- K-type thermocouple millivolt source
- K-type thermocouple wire
- PC with the following minimum requirements:

  - Windows XP Service Pack 2 (32-bit)
  - USB Type-A port
  - Processor rated at 1 GHz or faster
  - 512 MB RAM and 500 MB available hard disk space

Hardware Setup
--------------

- The EVAL-CN0354-PMDZ (CN-0354 board) connects to the :adi:`SDP-I-PMOD
  <eval_sdp-pmod>` interposer board via the 120-pin connector at **J3** and is
  connected to the :adi:`EVAL-SDP-CB1Z` (SDP-B board) via the 120-pin
  connector at **J4**.
- The EVAL-CFTL-6V-PWRZ +6 V power supply powers the CN-0354 board via the DC
  barrel jack of the SDP-PMOD interposer board at **J1**.
- The :adi:`EVAL-SDP-CB1Z` (SDP-B board) connects to the PC via the USB cable.
- The reference thermocouple connects to the CN-0354 board via the
  thermocouple connector at **+TREF-** and the tip of the thermocouple must be
  placed as close to **U4** as possible.
- The thermocouples may be placed at any of the four channels of **P2**.

.. note::

   If not all four thermocouple channels will be used, it is recommended to
   short out the positive and negative terminal of any unused channel at the
   isothermal block **P2** in order to avoid the :adi:`AD8495` from railing.

Software Setup
--------------

Installing the Software
~~~~~~~~~~~~~~~~~~~~~~~~

#. Extract the file **CN0354 SDP Eval Software.zip** and run **setup.exe**.

   .. note::

      It is recommended that you install the CN0354 Evaluation Software to the
      default directory path ``C:\Program Files\Analog Devices\CN0354\`` and all
      National Instruments products to ``C:\Program Files\National Instruments\``.

   .. figure:: software_install_welcome.png
      :align: center

      CN0354 Evaluation Software installer welcome screen

#. Click **Next** to view the installation review page.

   .. figure:: software_install_review.png
      :align: center

      Installation review page

#. Click **Next** to start the installation.

   .. figure:: software_install_progress.png
      :align: center

      Installation progress

#. Upon completion of the CN0354 Evaluation Software installation, the
   installer for the ADI SDP Drivers will execute.

   .. note::

      It is recommended that you close all other applications before clicking
      **Next**. This will make it possible to update relevant system files
      without having to reboot your computer.

   .. figure:: sdp_driver_install_welcome.png
      :align: center

      ADI SDP Drivers installer

#. Press **Next** to set the installation location for the SDP Drivers.

   .. note::

      It is recommended that you install the drivers to the default directory
      path ``C:\Program Files\Analog Devices\SDP\Drivers``.

   .. figure:: sdp_driver_install_location.png
      :align: center

      SDP Drivers installation location

#. Press **Next** to install the SDP Drivers and complete the installation of
   all software. Click **Finish** when done.

Using the Evaluation Software
~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~

Software Control and Indicator Descriptions
^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^

.. figure:: gui_start_sampling.png
   :align: center

   CN0354 Evaluation Software -- Start Sampling tab

#. **Start Sampling** -- When pressed, the SDP-B board makes a USB connection
   to the CN-0354 board automatically. Once a USB connection is established,
   the software starts sampling in real-time.
#. **Configure System** -- When pressed, the system is configured based on the
   preferences in the System Configuration tab.

   .. note::

      Default values are provided in the System Configuration tab so the
      software can run even without user-specified preferences.

#. **Save Data** -- When pressed, the software saves the data collected to a
   tab-delimited ASCII spreadsheet file.
#. **Control Tabs:**

   - *Start Sampling* -- Brings the data collection chart to the front.
   - *Configure System* -- Brings the system configuration settings to the
     front.
   - *SDP Board Information* -- Brings the SDP board revision information to
     the front.

#. **Thermocouple Channels Temperature Numerical Indicator** -- Displays the
   current temperature measurement in each active channel in degrees Celsius.
#. **Channel Select Checkboxes** -- Allow setting the active thermocouple
   channels.
#. **Chart Controls** -- Allow the user to zoom-in, zoom-out, and pan through
   the data collected.
#. **System Status String and LED Indicators** -- Display the current state of
   the software (Inactive, Busy, or Error).

.. figure:: gui_system_configuration.png
   :align: center

   CN0354 Evaluation Software -- System Configuration tab

9. **Temperature Range Dropdown Select** -- Used to change the temperature
   range of the system. The default value is 5 deg C to 880 deg C.
#. **Header Setting Tab Display** -- Shows the proper P1 header setting. The
   red box indicates where the shunt should be placed.

   .. important::

      Every time the temperature range is changed in the software, a
      corresponding change should be done in the hardware setting as well.
      Specifically, the **P1** header should have the correct setting as shown
      in the display tab of the Adjust Temperature Range section of the System
      Configuration tab.

#. **Low Calibration Temperature Numerical Control** -- Used to modify the low
   calibration temperature setpoint. The greyed out value on the right is the
   millivolt equivalent based on the NIST Table for Type K thermocouple. The
   default value is 5 deg C.
#. **High Calibration Temperature Numerical Control** -- Used to modify the
   high calibration temperature setpoint. The greyed out value on the right is
   the millivolt equivalent based on the NIST Table for Type K thermocouple.
   The default value is 880 deg C.
#. **Start Calibration** -- Calibrates the system using the values populated in
   the calibration temperature numerical controls.
#. **Calibration Select Dropdown Menu** -- Specifies whether to **Ignore
   Calibration** (software ignores the calibration factors) or to **Use
   Calibration** (software factors in the calibration values in the temperature
   calculations).
#. **Update Rate Radio Buttons** -- Used to change the output word rate of the
   :adi:`AD7787`. The default value is 16 Hz.
#. **Buffer Select Radio Buttons** -- Used to set the mode of the
   :adi:`AD7787`. The default value is Unbuffered Mode.

   - *Buffered Mode* -- Turns the on-chip analog input channel buffer of the
     :adi:`AD7787` **ON**.
   - *Unbuffered Mode* -- Turns the on-chip analog input channel buffer of the
     :adi:`AD7787` **OFF**.

#. **Burnout Current Radio Buttons** -- When enabled, the 100 nA current
   sources in the signal path are activated.

   .. important::

      The burnout currents can be enabled only when the buffer is active.

Saving Data to a Spreadsheet File
^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^

#. Establish a USB connection link.
#. Start sampling.
#. Click the **Save Data** button.
#. The software automatically opens the Excel spreadsheet file with the data.

.. note::

   The data is exported to an Excel spreadsheet file with columns labeled
   accordingly.
