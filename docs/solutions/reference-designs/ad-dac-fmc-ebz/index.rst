AD-DAC-FMC-EBZ
==============

Reference HDL design for JESD204B based DAC evaluation boards.

The :git-hdl:`dac_fmc_ebz <projects/dac_fmc_ebz>` is a generic reference
design that can be configured to support one of the following JESD204B based
DAC evaluation boards.

Block Diagram
-------------

The data path and clock domains are depicted on the below diagram.

.. figure:: dac_fmc_ebz_1.svg
   :align: center

   DAC FMC EBZ Data Path and Clock Domains

The data to be sent to DAC can have multiple sources:

DMA Source
~~~~~~~~~~

In case of high sample rates where the required data rate exceeds the PS-PL
interface available throughput, the data is transmitted in a loop from a local
buffer (dac_fifo) which is loaded once with the DMA from the PS DDR.
For lower sample rates the DAC FIFO can be placed in bypass mode, in this case
the DMA must stream the data from the PS memory.

DDS Source
~~~~~~~~~~

For each DAC channel a tone is generated by a
:git-hdl:`DDS core <docs/regmap/adi_regmap_dac.txt>`.

PRBS Source
~~~~~~~~~~~

For each DAC channel one of the following PN sequence can be selected: PN7,
PN15, inverted PN7, inverted PN15.

Software Defined Pattern Source
~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~

For each channel software can set the values which will be driven to the DAC.

Configuring the Design
----------------------

Before building the reference design the
:git-hdl:`config.tcl <projects/dac_fmc_ebz/common/config.tcl>` file must be
updated with the desired DAC device and JESD operation mode. The block design
will scale according the selected mode.

.. code-block:: tcl

   set device    AD9172
   set mode      04

A list of the supported DAC devices and JESD operation modes can be found in
the :git-hdl:`config.tcl <projects/dac_fmc_ebz/common/config.tcl>` file and
tables below. To find out more on the JESD operation modes consult the
datasheet of the selected DAC.
