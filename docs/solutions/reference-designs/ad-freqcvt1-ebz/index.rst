AD-FREQCVT1-EBZ
================

The :adi:`AD-FREQCVT1-EBZ` is a frequency up/down converter add-on board for
the :ref:`FMCOMMS3/4/5 <ad-fmcomms3-ebz>` boards which contain the
:adi:`AD9361`. The purpose of the AD-FREQCVT1-EBZ is to extend the AD9361
operating range down to 1 MHz through frequency up and down conversion.

.. image:: freqcvt1_top.jpg
   :align: center
   :width: 600

Introduction
------------

The AD-FREQCVT1-EBZ module is comprised of the :adi:`AD8342` mixer, the
:adi:`ADL5530` LNA, two :adi:`ADF4351` synthesizers, and power management
components. It is controlled via the PMOD port on the FPGA carrier board.

For additional information or to purchase the AD-FREQCVT1-EBZ, visit the
:adi:`AD-FREQCVT1-EBZ product page <AD-FREQCVT1-EBZ>`.

Features
~~~~~~~~

- Add-on to the :ref:`FMCOMMS3/4/5 <ad-fmcomms3-ebz>` family
- Powered from the FMCOMMS board
- Provides one channel down-conversion transmit and one channel up-conversion
  receive
- Extends AD9361 frequency range down to 1 MHz

Applications
~~~~~~~~~~~~

- Electronic test and measurement equipment
- General-purpose software radios
- Radar systems
- Ultra wideband satellite receivers
- Signals intelligence (SIGINT)
- Point-to-point communication systems
- Automated test equipment

Functional Overview
-------------------

The AD-FREQCVT1-EBZ extends the frequency of operation for the
:ref:`FMCOMMS3/4/5 <ad-fmcomms3-ebz>` boards:

- Input signals in the frequency range of 1--100 MHz are up-converted to an
  output signal centered around 340 MHz to the FMCOMMS board.
- An output signal in the range of 1--100 MHz is generated by down-converting
  a 370 MHz signal from the FMCOMMS board.
- On-board programmable phase-locked loop frequency synthesizers generate the
  needed local oscillator signals used by the up- and down-converting mixers.

Local Oscillator
^^^^^^^^^^^^^^^^

The local oscillator used on this board is the :adi:`ADF4351`, which is
software programmable. Two ADF4351 instances provide the TX and RX LO signals.
The board provides a 25 MHz crystal as the reference for the ADF4351.

SPI Interface
^^^^^^^^^^^^^

The SPI signals are directly passed through to the FMC connector. A 12-pin
ribbon cable connects the PMOD header on the FPGA carrier board to the
AD-FREQCVT1-EBZ digital control connector (P2), implementing the SPI protocol
and general-purpose I/Os used to control the two on-board :adi:`ADF4351`
synthesizers.

Quick Start Guide
-----------------

The Quick Start Guide provides step-by-step instructions detailing how to

:xilinx:`ZC706 <products/boards-and-kits/ek-z7-zc706-g.html>` and
:ref:`AD-FMCOMMS3-EBZ <ad-fmcomms3-ebz>` platforms are used to demonstrate the

Prerequisites
~~~~~~~~~~~~~

- :xilinx:`ZC706 <products/boards-and-kits/ek-z7-zc706-g.html>` carrier board
  and power supply
- :ref:`AD-FMCOMMS3-EBZ <ad-fmcomms3-ebz>` (or FMCOMMS4/5) board
- :adi:`AD-FREQCVT1-EBZ` board
- 12-pin ribbon cable (included)
- 3-pin power cable (included)
- 2 male-to-male SMA cables
- SMA loopback cable (for testing)
- SD card with :doc:`Kuiper Linux </linux/kuiper/index>` image
- HDMI display, USB keyboard and mouse

Connecting to FMCOMMS3/4 and Cable Orientation
~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~

The AD-FREQCVT1-EBZ has four SMA connectors (TX_IN, RX_OUT, TX_OUT, RX_IN).
Two male-to-male SMA connectors are required to connect the AD-FREQCVT1-EBZ
board to the FMCOMMS3 platform.

To use the second channel of the FMCOMMS3, line up FREQCVT1 connectors RX_OUT
and TX_IN with FMCOMMS3 connectors TX2A and RX2A.

.. figure:: dsc00053.jpg
   :align: center
   :width: 600

   AD-FREQCVT1-EBZ mounted on FMCOMMS3, connected to ZC706

To use the first channel of the FMCOMMS3, line up FREQCVT1 connectors RX_OUT
and TX_IN with FMCOMMS3 connectors RX1A and TX1A. This requires flipping the
board upside down.

.. figure:: 12_pin_cable_orientation.jpg
   :align: center
   :width: 600

   12-pin ribbon cable orientation

The :xilinx:`ZC706 <products/boards-and-kits/ek-z7-zc706-g.html>` board
provides digital controls to the AD-FREQCVT1-EBZ board through the 12-pin
ribbon cable. Be careful to properly plug in both ends of the cable according
to the image above.

Power Cable Connection
~~~~~~~~~~~~~~~~~~~~~~

Power is brought to the AD-FREQCVT1-EBZ board through a 3-pin cable. The
connection is made at P1 (pin 1 = 3.3 V, pin 2 = GND, pin 3 = 12 V), which
sits directly between TX_IN and RX_OUT. The power originates from the FMCOMMS3
board, connector P301.

.. important::

   It is critical to properly plug in the power cable on both ends. Connector
   P301 on the FMCOMMS3 board is not populated and requires a 3-pin male
   header. The cable must be carefully plugged into P301 to ensure the correct
   voltage is applied. The connector on the AD-FREQCVT1-EBZ board (P1) is
   keyed.

Software
--------

Theory of Operation
~~~~~~~~~~~~~~~~~~~~

The AD-FREQCVT1-EBZ extends the frequency of operation for the
:ref:`FMCOMMS3/4 <ad-fmcomms3-ebz>`. Two on-board programmable phase-locked
loop frequency synthesizers (:adi:`ADF4351`) generate the needed local
oscillator signals used by the up- and down-converting mixers. Input signals
in the frequency range of 1--100 MHz are up-converted to an output signal
centered around 340 MHz (RX SAW BPF) to the FMCOMMS board. An output signal
in the range of 1--100 MHz is generated by down-converting a 370 MHz (TX SAW
BPF) signal from the FMCOMMS board.

The Analog Devices board support packages for the FMCOMMS3/4 design, on the
various supported FPGA carriers, include an additional SPI interface routed to
the PMOD board header.

Two ADF4351 driver instances control the external synthesizers. The IIO
Oscilloscope FMCOMMS2/3/4 plugin implements the external and internal LO
synthesizer tuning logic. When the external synthesizers are detected, an
additional checkbox in the plugin is displayed, allowing the user to enable or
disable the tuning logic. Once enabled, the RX/TX LO spin button switches its
range from 70 MHz--6 GHz to 1 MHz--100 MHz.

.. figure:: ad-freqcvt1-ebz.png
   :align: center

   IIO Oscilloscope with AD-FREQCVT1-EBZ up/down converter enabled

Environment Variables
~~~~~~~~~~~~~~~~~~~~~

Two environment variables allow the user to control the sign in the mixing
equation as well as the window size (span) where the signal is centered around
the SAW band-pass filter center frequency.

.. list-table::
   :header-rows: 1

   * - Environment Variable
     - Default
     - Description
   * - ``OSC_UPDN_FREQ_SPAN``
     - 2 MHz
     - Window size around SAW BPF center frequency
   * - ``OSC_UPDN_FREQ_MIX_SIGN``
     - Positive
     - Sign of the mixing equation

Production Testing
------------------

The production testing process verifies each AD-FREQCVT1-EBZ board for gross
errors. The test first checks the noise floor with no input signal to ensure it
is within threshold, then inputs test tones and verifies the response is within
expected bounds.

Creating a Test SD Card
~~~~~~~~~~~~~~~~~~~~~~~

#. Write the latest :doc:`Kuiper Linux </linux/kuiper/index>` image to an SD
   card.
#. Copy the ``BOOT.BIN`` and ``devicetree.dtb`` files from the
   ``zynq-zc706-adv7511-ad9361-fmcomms2-3`` folder into the boot partition.
#. Modify the card to run tests automatically on boot. Test scripts from the
   `linux_image_ADI-scripts <https://github.com/analogdevicesinc/linux_image_ADI-scripts>`__
   repository automate initializing the IIO Oscilloscope with the correct
   profile and environment. Update the desktop autostart launcher to execute
   the test script instead of IIO Oscilloscope.

Required Hardware
~~~~~~~~~~~~~~~~~

- :xilinx:`ZC706 <products/boards-and-kits/ek-z7-zc706-g.html>` carrier board
  and power supply
- HDMI monitor, USB keyboard and mouse
- SD test card (prepared above)
- :ref:`AD-FMCOMMS3-EBZ <ad-fmcomms3-ebz>` board
- :adi:`AD-FREQCVT1-EBZ` board
- 12-pin ribbon cable
- 3-pin power cable
- SMA loopback cable

Test Process
~~~~~~~~~~~~

#. Ensure the ZC706 is set up with the testing SD card, HDMI monitor, and
   keyboard/mouse connected and powered on.
#. Mount the AD-FREQCVT1-EBZ board on the TX2A/RX2A jacks of the
   AD-FMCOMMS3-EBZ board. Connect the power cable and signal cables as shown
   below.

   .. figure:: freqcvt1-testing-cables.jpg
      :align: center
      :width: 600

      AD-FREQCVT1-EBZ testing cable connections

#. Connect the RX_IN and TX_OUT jacks on the AD-FREQCVT1-EBZ board with the
   SMA loopback cable.
#. Power on the ZC706 board. The system boots into the ADI Linux environment
   and begins tests automatically.
#. When tests pass, a confirmation window is displayed. If tests fail, an error
   dialog appears instead. Confirm the result to halt Linux, then power off the
   carrier board.

   .. figure:: freqcvt1-tests-passed.png
      :align: center
      :width: 600

      AD-FREQCVT1-EBZ production test passed

#. Remove the AD-FREQCVT1-EBZ board and repeat the process for the next unit.
