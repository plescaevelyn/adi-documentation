EVAL-ADMX2001EBZ
================

Overview
-------------------------------------------------------------------------------

The :adi:`EVAL-ADMX2001EBZ` is an easy-to-use evaluation and development
breakout board for the :adi:`ADMX2001B` High-Performance Precision Impedance
Analyzer Measurement Module. It enables resistance measurements at DC and
impedance measurements from 0.2 Hz to 10 MHz with 18-bit acquisition channels.

Features:

- Impedance measurements from 0.2 Hz to 10 MHz
- 18-bit acquisition channels
- BNC connectors for standard LCR meter test probes and fixtures
- UART and SPI host interfaces
- Trigger and clock synchronization via SMA connectors
- Arduino-style headers compatible with SDP-K1
- Power jack accepts +5V to +12V input

Applications:

- Passive component characterization (R, L, C measurement)
- Impedance spectroscopy
- Material characterization
- LCR meter evaluation and development
