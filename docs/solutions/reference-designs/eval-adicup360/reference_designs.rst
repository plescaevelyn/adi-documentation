Reference Designs
=================

This chapter contains various reference designs available for the base board and
the various shields. Each sub-section describes the demo program, how to setup
the hardware, how to obtain the source code, and finally, how to import the
project in the Eclipse workspace and to run it.

The following reference designs are currently available:

-  :doc:`Blinking LEDs Demo </solutions/reference-designs/eval-adicup360/reference_designs/demo_blink>` - Shows the basic steps of creating a new project for the :doc:`EVAL-ADICUP360 base board </solutions/reference-designs/eval-adicup360/hardware/base_board>`, running and debugging the software.
-  :doc:`Command Line Interpreter Demo </solutions/reference-designs/eval-adicup360/reference_designs/demo_cli>` - A Command Line Interpreter (CLI) demo project for the :doc:`EVAL-ADICUP360 base board </solutions/reference-designs/eval-adicup360/hardware/base_board>`.
-  :doc:`Accelerometer Demo </solutions/reference-designs/eval-adicup360/reference_designs/demo_adxl362>` - Illustrates the functionality of the ADXL362 3-axes accelerometer. It works with the :doc:`EVAL-ADXL362-ARDZ Shield </solutions/reference-designs/eval-adicup360/hardware/adxl362>`.
-  :doc:`PMOD Accelerometer Demo </solutions/reference-designs/eval-adicup360/reference_designs/demo_pmodacl2>` - Illustrates the functionality of the ADXL362 3-axes accelerometer. It works with the `PmodACL2 from Digilent <http://store.digilentinc.com/pmod-acl2-3-axis-mems-accelerometer/>`_.
-  :doc:`Weigh Scale Demo </solutions/reference-designs/eval-adicup360/reference_designs/demo_cn0216>` - Weigh Scale measurement example for 4-/6-wire bridge sensors. It works with the :doc:`EVAL-CN0216-ARDZ Shield </solutions/reference-designs/eval-adicup360/hardware/cn0216>`.
-  :doc:`pH Measurement Demo </solutions/reference-designs/eval-adicup360/reference_designs/demo_cn0326>` - pH Measurement System with Temperature Compensation that works with the :adi:`EVAL-CN0326-PMDZ Pmod <cn0326>`.
-  :doc:`Data Acquisition for Input Current Demo </solutions/reference-designs/eval-adicup360/reference_designs/demo_cn0336>` - Handles data of the acquisition system for 4-20 mA inputs current that works with the :adi:`EVAL-CN0336-PMDZ Pmod <cn0336>`.
-  :doc:`RTD Temperature Measurement Demo </solutions/reference-designs/eval-adicup360/reference_designs/demo_cn0337>` - RTD temperature measurement example that works with the :adi:`EVAL-CN0337-PMDZ Pmod <cn0337>`.
-  :doc:`Carbon Dioxide Gas Detection Demo </solutions/reference-designs/eval-adicup360/reference_designs/demo_cn0338>` - Non Dispursive Infrared Gas Detection that works with the :doc:`EVAL-CN0338-ARDZ Shield </solutions/reference-designs/eval-adicup360/hardware/cn0338>`.
-  :doc:`Toxic Gas(CO) Detection Demo </solutions/reference-designs/eval-adicup360/reference_designs/demo_cn0357>` - Measuring toxic gases using electrochemical sensors that work with the :doc:`EVAL-CN0357-ARDZ Shield </solutions/reference-designs/eval-adicup360/hardware/cn0357>`.
-  :doc:`Universal Multichannel Thermocouple Measurement Demo (Digital) </solutions/reference-designs/eval-adicup360/reference_designs/demo_cn0391>` - Measure up to four(4) thermocouples of any type and combination using the :doc:`EVAL-CN0391-ARDZ Shield </solutions/reference-designs/eval-adicup360/hardware/cn0391>`.
-  :doc:`Universal Multichannel Thermocouple Measurement Demo (Analog) </solutions/reference-designs/eval-adicup360/reference_designs/demo_cn0394>` - Measure up to four(4) thermocouples of any type and combination using the :doc:`EVAL-CN0394-ARDZ Shield </solutions/reference-designs/eval-adicup360/hardware/cn0394>`.
-  :doc:`VOC Gas Detection Measurement Demo </solutions/reference-designs/eval-adicup360/reference_designs/demo_cn0395>` - Measure the level of Volatile Organic Compounds(VOC) in the air using the :doc:`EVAL-CN0395-ARDZ Shield </solutions/reference-designs/eval-adicup360/hardware/cn0395>`.
-  :doc:`Dual Toxic Gas Detection Demo </solutions/reference-designs/eval-adicup360/reference_designs/demo_cn0396>` - Measuring toxic gases using electrochemical sensors that work with the :doc:`EVAL-CN0396-ARDZ Shield </solutions/reference-designs/eval-adicup360/hardware/cn0396>`.
-  :doc:`Visible Light(RBG)Detection Demo </solutions/reference-designs/eval-adicup360/reference_designs/demo_cn0397>` - Measuring visible light intensity and spectrum concentration levels without optical filters or other mechanical components using the :doc:`EVAL-CN0397-ARDZ Shield </solutions/reference-designs/eval-adicup360/hardware/cn0397>`.
-  :doc:`Soil Moisture and pH Measurement Demo </solutions/reference-designs/eval-adicup360/reference_designs/demo_cn0398>` - Measures the moisture level and and pH value of different kinds of soil using the :doc:`EVAL-CN0398-ARDZ Shield </solutions/reference-designs/eval-adicup360/hardware/cn0398>`.
