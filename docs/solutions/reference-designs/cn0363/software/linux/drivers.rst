Linux Device Drivers
====================

EVAL-CN0363-PMDZ uses Linux drivers in the `IIO framework <https://wiki.analog.com/software/linux/docs/iio/iio>`_; it is possible to use the `IIO-Scope <https://wiki.analog.com/resources/tools-software/linux-software/iio_oscilloscope>`_ to access the raw sample data from the EVAL-CN0363-PMDZ. This is mainly intended to be used for debugging purposes.

Here is the link to `Linux device drivers <https://wiki.analog.com/resources/tools-software/linux-drivers-all>`_ for some Analog Devices products.

A detailed step on Zedboard driver installation for Debian is provided `here <https://github.com/PyHDI/zynq-linux>`_.
