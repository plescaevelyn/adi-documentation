EVAL-CN0363-PMDZ Linux software applications
============================================

The primary program to interact with the EVAL-CN0363-PMDZ is the `ADI Colorimeter application <https://wiki.analog.com/resources/tools-software/linux-software/colorimeter>`_. It can be used to gather, manage and compare processed sample data from the EVAL-CN0363-PMDZ.

Since the Linux drivers used for the EVAL-CN0363-PMDZ use the `IIO framework <https://wiki.analog.com/software/linux/docs/iio/iio>`_ it is possible to use the `IIO-Scope <https://wiki.analog.com/resources/tools-software/linux-software/iio_oscilloscope>`_ to access the raw sample data from the EVAL-CN0363-PMDZ. This is mainly intended to be used for debugging purposes.

All applications run directly on the ZED board itself.
