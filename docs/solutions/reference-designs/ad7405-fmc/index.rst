AD7405-FMC User Guide
=====================

Introduction
------------

The :adi:`AD7405` is a second-order, high performance sigma-delta modulator
that converts an analog input signal into a high speed, single-bit digital
data stream. It is designed for isolated current and voltage measurement.

-----------------
