AD-FMCOMMS5-EBZ
===============

Dual AD9361 MIMO Software Defined Radio Board.

The :adi:`AD-FMCOMMS5-EBZ` is an FMC board featuring two :adi:`AD9361` devices,
demonstrating how to design a platform based on multiple devices for MIMO
applications requiring multi-channel RF synchronization.

This board requires a carrier with dual FMC connectors and is compatible with
the ZC702 and ZC706 Zynq-based Xilinx boards.
