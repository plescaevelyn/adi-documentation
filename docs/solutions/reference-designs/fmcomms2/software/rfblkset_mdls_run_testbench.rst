How to run the AD9361 Receive Testbench?
===============================================================================

- Open the Simulink Library browser.
- In the library browser, find RF Blockset Models for Analog Devices
  Transceivers.
- Under AD9361 Testbenches, find ad9361_RX and double click it to
  open the testbench.

.. image:: ../images/lib_browser.png
   :align: center
   :width: 600

- Double-click Manual Switch block to toggle between continuous wave
  (CW) or LTE source input.

.. image:: ../images/manual_switch.png
   :align: center
   :width: 600

- Set the Simulation Stop Time to 0.005 s.

.. image:: ../images/simulation_time.png
   :align: center
   :width: 600

- Run the simulation using the Play button.
