AD-FMCOMMS2-EBZ / AD-FMCOMMS3-EBZ / AD-FMCOMMS4-EBZ / AD-FMCOMMS5-EBZ Linux support
===================================================================================

.. image:: ../images/tux.png
   :align: right

Analog Devices provides full Linux support for the
AD-FMCOMMS2-EBZ/AD-FMCOMMS3-EBZ/AD-FMCOMMS4-EBZ/AD-FMCOMMS5-EBZ. Depending on
which FPGA platform you are using the steps to perform to setup the system
slightly differ:

-  :doc:`microblaze </solutions/reference-designs/fmcomms2/software/linux/microblaze>` - compiling/building linux kernel for Microblaze eval boards
-  `KC705 (Microblaze) <https://wiki.analog.com/resources/tools-software/linux-drivers/iio-transceiver/ad9361>`_
-  :doc:`ZC702, ZC706, ZED (Zynq) </solutions/reference-designs/fmcomms2/software/linux/zynq>`

Descriptions of the individual Linux device drivers for the different parts on
the AD-FMCOMMS2-EBZ / AD-FMCOMMS3-EBZ / AD-FMCOMMS4-EBZ / AD-FMCOMMS5-EBZ can be
found at:

-  `AD9361: RF Agile Transceiver <https://wiki.analog.com/resources/tools-software/linux-drivers/iio-transceiver/ad9361>`_
-  `AD7291: 8-Channel, I2C, 12-Bit SAR ADC with Temperature Sensor <https://wiki.analog.com/resources/tools-software/linux-drivers/iio-adc/ad7291>`_
