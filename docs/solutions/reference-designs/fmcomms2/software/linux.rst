AD-FMCOMMS2-EBZ / AD-FMCOMMS3-EBZ / AD-FMCOMMS4-EBZ / AD-FMCOMMS5-EBZ Linux support
===================================================================================

.. image:: ../images/tux.png
   :align: right

Analog Devices provides full Linux support for the
AD-FMCOMMS2-EBZ/AD-FMCOMMS3-EBZ/AD-FMCOMMS4-EBZ/AD-FMCOMMS5-EBZ.
Depending on which FPGA platform you are using the steps to
perform to setup the system slightly differ:

- :ref:`Microblaze <linux-kernel microblaze>` - compiling/building
  linux kernel for Microblaze eval boards
- :ref:`Zynq <linux-kernel zynq>`
- :ref:`Zynqmp <linux-kernel zynqmp>`

Descriptions of the individual Linux device drivers for the
different parts on the AD-FMCOMMS2-EBZ / AD-FMCOMMS3-EBZ /
AD-FMCOMMS4-EBZ / AD-FMCOMMS5-EBZ can be found at:

- :external+linux:doc:`AD9361: RF Agile Transceiver
  <drivers/iio-transceiver/ad9361>`
- :external+linux:doc:`AD7291: 8-Channel, I2C, 12-Bit SAR ADC
  with Temperature Sensor <drivers/iio-adc/ad7291>`
