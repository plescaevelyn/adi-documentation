EVAL-ADXL357BZ
===============

Low Noise 3-Axis MEMS Accelerometer.

Overview
--------

The EVAL-ADXL357B is a simple evaluation board for quick evaluation of the
:adi:`ADXL357B` low-noise, low-power, 3-axis MEMS accelerometer. The board's
stiffness and small size minimize the effect on both the system and acceleration
measurements.

The ADXL357B provides selectable measurement ranges (±10g, ±20g, ±40g) with
industry-leading noise density, 1 mg/LSB resolution on the ±10g range, and
minimal offset drift over temperature for precision applications with minimal
calibration. Communication is via SPI or limited I2C interface.

Required Equipment
------------------

- EVAL-ADXL357B evaluation board
- Host platform with SPI or I2C interface
- Power supply: 1.7--3.6 V

Hardware Setup
--------------

- Two sets of spaced vias for population of 5-pin headers.
- Board can be easily attached to a prototyping board or PCB.
- Small form factor minimizes impact on system and measurement.

Software Setup
--------------

Device drivers are available via the no-OS driver framework and Linux IIO
drivers for integration with host platforms.
