Overview
--------

The :adi:`EVAL-ADIS2` is iSensor's second-generation, PC-based evaluation system that supports communication with most ADIS16xxx products, using the "breakout board" evaluation tools for each product. The :adi:`EVAL-ADIS2` design includes onboard voltage regulation and high-speed USB data transfer built upon the :adi:`ADSP-BF527` DSP.
