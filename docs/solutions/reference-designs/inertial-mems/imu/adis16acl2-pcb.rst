ADIS16ACL2/PCB WIKI GUIDE
=========================

OVERVIEW
--------

The ADIS16ACL2/PCBZ is the breakout board for the ADIS16223 and ADIS16227
vibration sensing solutions. This breakout board function helps connect these
two products to the EVAL-ADIS2 evaluation system. It also helps connect these
products to existing embedded processor platforms.

.. image:: ../images/adis16acl2_pcb_web_000.png
   :width: 600

ORDERING
--------

To order this breakout board, please visit the :adi:`ADIS16ACL2/PCBZ's web page <en/design-center/evaluation-hardware-and-software/evaluation-boards-kits/EVAL-ADIS16ACL2.html>`

ADIS16223/ADIS16227 Connection
------------------------------

Press the ADIS16223/7 connector onto J2 (14-pin socket).

.. image:: ../images/adis16acl2_pcb_web_003.png
   :width: 450

EVAL-ADIS2 Connection
---------------------

The EVAL-ADIS2 provides PC-based evaluation support for the ADIS16223 and
ADIS16227. J1 on the EVAL-ADIS2Z and the ADIS16ACL2/PCBZ use the same pin
assignments, so connecting them requires a 16-pin ribbon cable.
