Evaluating the ADXL362
======================

*Useful links:* ADXL362 :adi:`product page <adxl362>` \| :adi:`datasheet <static/imported-files/data_sheets/ADXL362.pdf>`.

Several options exist for evaluating the ADXL362:

-  **View real-time data on a complete evaluation system.**

   -  EVAL-ADXL362Z-MLP Evaluation System :doc:`User Guide </solutions/reference-designs/inertial-mems/accelerometers/adxl362/eval-adxl362z-mlp>`
   -  :adi:`Order <en/mems-sensors/mems-accelerometers/adxl362/products/ADXL362Z/eb.html#buy>` the EVAL-ADXL362Z-MLP Evaluation System

-  **Use an existing platform to log acceleration data. Post-process the data to develop algorithms. Use these algorithms in your own system.**

   -  Datalogger :doc:`Quick-Start Guide </solutions/reference-designs/inertial-mems/accelerometers/adxl362/eval-adxl362z-db>`
   -  :adi:`Order <en/mems-sensors/mems-accelerometers/adxl362/products/EVAL-ADXL362/eb.html#buy>` a Datalogger Board (same as the Development Board)

-  **Use an existing, reprogrammable platform to develop firmware. Port the entire platform -- schematics, layout, and firmware -- into your own system.**

   -  Development Board :doc:`Quick-Start Guide </solutions/reference-designs/inertial-mems/accelerometers/adxl362/eval-adxl362z-db>`

      -  :adi:`Order <en/mems-sensors/mems-accelerometers/adxl362/products/EVAL-ADXL362/eb.html#buy>` a Development Board (same as the Datalogger Board)

-  **Integrate a breakout board into an existing system via wires or breakout board.**

   -  Breakout Board User Guide

      -  :adi:`Order <en/mems-sensors/mems-accelerometers/adxl362/products/EVAL-ADXL362Z/eb.html#buy>` a breakout board

-  **Integrate the IC into your own PCB.**

   -  Order the ADXL362 from :adi:`Analog Devices <adxl362#buy>` \| `Arrow <http://parts.arrow.com/item/detail/analog-devices/adxl362bccz>`_ \| `Avnet <https://avnetexpress.avnet.com/store/em/EMController?langId=-1&storeId=500201&catalogId=500201&N=0&action=products&term=adxl362>`_ \| `Digikey <https://www.digikey.com/EVAL-ADXL362-ARDZ>`_ \| Element14(coming soon) \| Sparkfun (coming soon)

      -  Schematics and layout for integration into your PCB
