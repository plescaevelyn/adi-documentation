Quick start
================================================================================

The quick start guides provide simple step by step instructions on how to do
an initial system setup for the :adi:`EVAL-CN0363-PMDZ <CN0363>`
board on the ZED FPGA development board. In these guides, we will discuss how
to program the SD card and boot a Linux distribution.

.. toctree::

   On ZED <zynq>

Supported carriers
--------------------------------------------------------------------------------

The :adi:`EVAL-CN0363-PMDZ <CN0363>` connects via a PMOD interface
to the carrier board.

.. list-table::
   :header-rows: 1

   - - FPGA board
     - EVAL-CN0363-PMDZ
   - - `ZED Board <https://www.avnet.com/americas/products/avnet-boards/avnet-board-families/zedboard/>`_, Rev C or later
     - PMOD JA
