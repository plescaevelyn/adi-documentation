EVAL-CN0363-PMDZ Linux software applications
============================================

The primary program to interact with the EVAL-CN0363-PMDZ is the :doc:`ADI Colorimeter application </wiki-migration/resources/tools-software/linux-software/colorimeter>`. It can be used to gather, manage and compare processed sample data from the EVAL-CN0363-PMDZ.

Since the Linux drivers used for the EVAL-CN0363-PMDZ use the :doc:`IIO framework </wiki-migration/software/linux/docs/iio/iio>` it is possible to use the :doc:`IIO-Scope </wiki-migration/resources/tools-software/linux-software/iio_oscilloscope>` to access the raw sample data from the EVAL-CN0363-PMDZ. This is mainly intended to be used for debugging purposes.

All applications run directly on the ZED board itself.
