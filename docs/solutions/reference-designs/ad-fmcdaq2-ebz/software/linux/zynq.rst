AD-FMCDAQ2-EBZ/AD-FMCDAQ3-EBZ on Zynq ZC706
==============================================

The image for the AD-FMCDAQ2-EBZ/AD-FMCDAQ3-EBZ FMC Card on the ZC706 can be
created by following the
:doc:`Zynq images directions </linux/kuiper/index>`.

After that is complete, and running, you should see the IIO scope.
