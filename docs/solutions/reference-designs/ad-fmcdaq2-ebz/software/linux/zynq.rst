AD-FMCDAQ2-EBZ/AD-FMCDAQ3-EBZ on Zynq ZC706
===============================================================================

The image for the :adi:`AD-FMCDAQ2-EBZ`/:adi:`AD-FMCDAQ3-EBZ` FMC Card on the
ZC706 can be found, and created by following the directions at
:external+kuiper:doc:`Kuiper Linux <index>`.

After that is complete, and running, you should see the IIO scope.
