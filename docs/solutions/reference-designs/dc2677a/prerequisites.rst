Prerequisites
===============================================================================

What you need, depends on what you are trying to do. As a minimum, you need to
start out with:

Hardware prerequisites
-------------------------------------------------------------------------------

#. The DC2677A evaluation board: :adi:`DC2677A`
#. An FPGA carrier platform: `Cyclone V SoC Development Kit
   <https://www.arrow.com/en/products/sockit/arrow-development-tools>`_
#. Power supply: +16V/+18V and -16V/-18V for DC2677A
#. Power supply: 12V for carrier board
#. An SD card with at least 16GB of memory; see :dokuwiki:`Kuiper Linux
   <resources/tools-software/linux-software/kuiper-linux>`
#. USB cable (Male Mini-B port for Cyclone V SoC Development Kit and Male A port
   for PC)

Software prerequisites
-------------------------------------------------------------------------------

#. :ref:`libiio` — IIO library for interfacing with the LTC235x driver

.. note::

   :adi:`ADI <>` does not offer FPGA carrier platforms for sale or loan; getting
   one yourself is the normal part of development or evaluation.
