HMC980 Configuration Calculator
===============================

This calculator can be used to calculate all of the external component values
that are required to configure current settings, voltage ranges and alarm levels
for HMC980

:download:`https://wiki.analog.com/_media/resources/eval/user-guides/active-bias-controllers/hmc980.zip`

.. figure:: https://wiki.analog.com/_media/resources/eval/user-guides/active-bias-controllers/hmc980_configuration_calculator.png
   :width: 400px

Verified Amplifier Designs using the HMC980
===========================================

.. important::

   :red:`Please note that while these tables are extensive, they are not exhaustive. If a given RF amplifier is not included in this list, that does not mean that it cannot be actively biased using the :adi:`HMC980 <hmc980lp4e>`.
   Follow the
   :adi:`HMC980 Datasheet <media/en/technical-documentation/data-sheets/hmc980.pdf>`
   for design guidelines.`
