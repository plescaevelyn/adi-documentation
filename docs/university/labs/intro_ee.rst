General Background Information
-------------------------------------------------------------------------------

Before beginning these lab activities, you should be familiar with either the
ADALM1000/ALICE or ADALM2000/Scopy system. Review the quick start guides and
basic measurement tutorials listed in the Lab Preparation section.

Things to Think About
-------------------------------------------------------------------------------

For successful circuit-building exercises, follow these steps:

1. **Measure component values** - Record actual resistance, capacitance, and
   inductance values before construction. Components have tolerances that affect
   circuit behavior.

2. **Draw schematic diagrams** - Create a clear schematic of the circuit you
   plan to build.

3. **Build on breadboards** - Construct your circuit on a solderless breadboard,
   following your schematic.

4. **Verify before powering** - Double-check your wiring against the schematic
   before applying power.

5. **Perform mathematical analysis** - Calculate expected voltages and currents.
   LTspice simulation can assist with complex circuits.

6. **Measure carefully** - Take voltage and current measurements systematically
   and record all data.

7. **Troubleshoot errors** - If measured values differ substantially from
   calculations, investigate the cause.

Start with simple circuits and incrementally add complexity as you verify each
stage works correctly.

The Good Lab Report
-------------------------------------------------------------------------------

Objectives
~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~

Provide a brief one-paragraph summary of the objectives of the lab experiments,
including the steps to be performed and expected test results.

Pre Lab
~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~

Describe the preparation done prior to the lab, including any calculations,
simulations, or expected results. This is often formalized in lab worksheets.

Equipment and Materials
~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~

List all equipment used (with make and model numbers) and provide a complete
list of components from the lab worksheet.

Methods
~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~

Include detailed descriptions of each experiment performed, covering:

* Circuit descriptions
* Test setup procedures
* Data acquisition methods

Support descriptions with diagrams and schematics where appropriate.

Results and Discussion
~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~

Include all test results, organized in the order performed. Present data in a
logical, concise format such as graphs or tables with proper labeling. Discuss
any apparent errors or unexpected results.

Conclusions
~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~

Summarize your results and lessons learned from problems and mistakes along the
way. Include comments on potential next steps or further investigations.

Measurement Errors
~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~

Understanding the difference between accuracy and precision is essential for
interpreting measurement results.

**Accuracy** is the difference between the measured value and the true value
of a quantity. It can be expressed as:

* **Absolute error**: The direct difference between measured and true values
* **Relative error**: The ratio of absolute error to true value (often as percentage)

**Precision** relates to the smallest difference between measured values that
can be distinguished by the instrument.

An instrument can be precise but inaccurate (well-calibrated resolution but
systematic offset). Relative errors often matter more than absolute errors
depending on the magnitude of the measured quantity.

Component Selection Notes
-------------------------------------------------------------------------------

Transistors
~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~

Recommended transistor types for these labs:

**Bipolar Junction Transistors (BJTs):**

* NPN: 2N3904
* PNP: 2N3906
* Asian equivalents: 2SC1815 (NPN), 2SA1015 (PNP)

**Transistor Arrays:**

* LM3045, LM3046, LM3086 (National Semiconductor)
* CA3045, CA3046, CA3083 (Intersil)

**MOSFETs:**

* NMOS: 2N7000
* MOS Arrays: ALD1106, ALD1107 (Advanced Linear Devices)
* CMOS: CD4007C

.. warning::

   Not all transistors share the same terminal designations (pinouts). Always
   check the manufacturer's datasheet before using a transistor. Verify terminal
   identification using a multimeter's diode check function.

Diodes and Voltage References
~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~

Silicon junction voltage drops vary with conditions:

* Typical silicon BJT base-emitter drop: ~0.65V
* Standard silicon diode forward voltage: ~0.6V
* At elevated temperatures, VBE drops toward ~0.5V
* Bandgap reference voltage: ~1.22V (extrapolated VBE at absolute zero)
