DE10-Nano

Hardware Setup

  1. Mount the EVAL-CN0579-ARDZ onto the DE10-Nano via the Arduino headers.

  2. Connect a Mini-USB cable from the DE10-Nano’s UART port to the host PC.

  3. Connect an Ethernet cable from the DE10-Nano to the host PC or network switch.

  4. Connect a signal generator to the EVAL-CN0579-ARDZ analog inputs using SMA cables.

  5. Insert the microSD card with the ADI Kuiper Linux image into the microSD card slot on the DE10-Nano.

  6. Open a serial terminal of your choice (e.g. PuTTY, Tera Term) on the host PC configured to 115200 baud, 8N1.

  7. Plug in the DE10-Nano power supply.

  8. Wait for the system to boot. Boot messages will appear in the serial terminal.

Software Setup

Preparing the SD Card

  1. Download and flash the ADI Kuiper Linux image to a 16 GB (or larger) microSD card. Follow the Use Kuiper Image guide for instructions.

  2. Mount the FAT32 BOOT partition of the SD card and copy the following files to its root:

    • socfpga_cyclone5_common/zImage

    • <target_project>/u-boot.scr

    • <target_project>/socfpga.dtb

    • <target_project>/soc_system.rbf

  3. Create the extlinux folder at the root of the BOOT partition and place the extlinux.conf file inside it:

    mkdir -p /media/$USER/BOOT/extlinux
cp socfpga_cyclone5_common/extlinux.conf /media/$USER/BOOT/extlinux/extlinux.conf

  4. Write the U-Boot SPL image to the third SD card partition (raw, not a filesystem):

    dd if=<target_project>/u-boot-with-spl.sfp of=/dev/sdX3 bs=64k && sync

    Replace /dev/sdX3 with the correct device node for your SD card (e.g. /dev/sdd3). Use lsblk to identify it.

  5. Safely unmount the SD card and insert it into the microSD card slot on the DE10-Nano.

Install IIO Oscilloscope

Download and install one of the following tools on your PC:

  • IIO Oscilloscope - the classic IIO-based waveform viewer

  • Scopy - a more modern alternative with a built-in IIO plugin that can connect to the board over the network

Connecting to the Board

Open IIO Oscilloscope and connect to the board using its IP address over the network. Use the ifconfig command on the serial terminal to find the board’s IP address.

Configure the device settings as needed from the configuration panel.

Data Capture

Use the DMM panel to read instantaneous ADC channel values.

Switch to the oscilloscope view to capture waveforms in the time domain.

For frequency-domain analysis, switch to the FFT view.

Useful Commands

Check the network interface and assigned IP address:

~$

ifconfig

List all IIO devices detected by the kernel:

~$

iio_info -s

Power off the board safely:

~$

poweroff

PyADI-IIO

PyADI-IIO is a Python abstraction layer for ADI hardware with IIO drivers. It provides a device-specific API for the CN0579 that makes it easy to stream data directly into Python without dealing with the low-level IIO interface.

Follow the PyADI-IIO page for installation instructions, then run the CN0579 example script connecting to the board’s IP address.
